## 场景2：送药

- **地点**：茅草屋内，新手村边缘
- **时间**：清晨
- **出场角色**：李长风、林婉儿
- **核心动作**：林婉儿推门而入，为主角送药；两人在对话中建立初步关系；主角注意到自己手背上浮现的神秘剑纹
- **对话要点**：
  1. 林婉儿关切询问主角伤势（"你醒了？感觉怎么样？"）
  2. 主角试探性回应，确认自己穿越的事实（通过林婉儿的外貌、服饰、用语判断）
  3. 林婉儿提到主角受伤的原因（"那天晚上你从后山回来，浑身是伤……"）——不详细说明，为主角留下悬念
  4. 林婉儿离开前叮嘱好好休息，暗示会再来
- **过渡方式**：林婉儿离去，门关上后，门外传来隐约的说话声

### 场景细节要求

1. **林婉儿首次亮相的外貌描写**：
   - 素色衣裙、清秀面容、眼神灵动但带着一丝不易察觉的深邃
   - 暗示身份不简单，为F005-1伏笔做铺垫

2. **林婉儿的言行要"过于镇定"**：
   - 见到主角苏醒没有表现出过度的惊讶，仿佛早有预料
   - F005-1伏笔：穿越者见到穿越者异常镇定

3. **手背剑纹的描写**：
   - 青黑色纹路，形似一柄古剑的轮廓，隐隐发烫
   - 主角不确定是原主就有的还是穿越后出现的

4. **对话中要体现主角的试探心理**：
   - 利用游戏经验判断林婉儿是否是NPC
   - 发现她的反应过于真实

5. **重要约束**：
   - 林婉儿出场后，第2-11章将退场不出现
   - 对话要有分寸感，不要过度展开关系
   - 不要写主角下床活动或探索屋外

## 角色设定

### 李长风（主角）

**性格特征**：
- 冷静、果断、重情义、有野心、善于分析
- 遇到问题先分析，不急于表态
- 关键时刻果断，不拖泥带水

**说话风格**：
- 话不多，但每句都有分量
- 遇到问题先分析，不急于表态
- 不轻易说"我做不到"

**典型对话示例**：
```
"剑不利，人先死。"
"让我看看你有几斤几两。"
"你说的不无道理，但我有我的剑。"
```

### 林婉儿（女主）

**性格特征**：
- 聪明、冷静、外冷内热、心思缜密
- 表面柔弱，实则心机深沉
- 善于用问句引导对方
- 当前处于"警惕模式"：因穿越者身份需要隐藏，对陌生人保持高度戒备
- 温和是伪装/习惯，不是本性冷漠

**说话风格**：
- 说话温和，但内藏锋芒
- 善于用问句引导对方
- 危急时刻冷静得出奇
- 不说废话，不称呼亲昵称呼

**典型对话示例**：
```
"你的剑法退步了。"
"不是关心你，只是不想失去一个好打手。"
"你确定要这样做？"
```

**感情温度计**：0→15（送药+手背剑纹+系统异常试探）
