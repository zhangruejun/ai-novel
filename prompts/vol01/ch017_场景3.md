## 场景三：危机与收获
- 地点：地下大厅→石阶→地面（石屋后壁裂缝）
- 时间：Day 6 中午
- 出场角色：李长风、林婉儿
- 崩塌触发（简洁，不要拖沓）：
  1. 李长风将玉简收入贴身衣袋的瞬间，大厅顶部的裂缝扩大
  2. 不是自然崩塌——是阵法被触发后的连锁反应
  3. 瘴气污染了原本的稳定结构，遗迹的能量平衡被打破
  4. 碎石从顶部落下，砸在地面上发出沉闷的响声
- 撤离过程（动作主导，紧凑）：
  1. 李长风拉着林婉儿往石阶方向跑
  2. 右腿每落地一次都传来剧烈刺痛，但他没有停
  3. 林婉儿跟在身后，手背剑纹烫得厉害，但她咬紧牙关没有出声
  4. 石阶上的壁画开始脱落，凿刻的痕迹在崩塌中碎裂
  5. 跑到第十五六级时，一侧石壁塌陷，碎石滚落，险些堵住通路
  6. 两人侧身挤过狭窄的缝隙，继续往上跑
- 关键物品（确认携带）：
  1. 铜牌（已有一件，贴身衣袋）
  2. 玉简（新获得，含云崖子遗言，贴身衣袋）
  3. 遗骸旁储物袋中的残破笔记（只来得及抓了几页，纸张脆化但字迹还能辨认，拿在手中或塞入衣袋）
- 回到地面：
  1. 两人从石屋后壁裂缝爬出，回到地面
  2. 石屋在身后发出沉闷的响声
  3. 裂缝扩大，石阶入口被塌陷的石块封住
  4. 遗迹彻底封闭
- 瘴气变化（微妙，不是完全消失）：
  1. 回到地面后，李长风注意到周围的瘴气有变化
  2. 比进入石阶之前淡了一些
  3. 颜色从灰绿色变为浅灰色
  4. 浓度降低了约三分之一
  5. 像是遗迹封闭后，瘴气的"源头"被削弱了
- 新悬念（林婉儿剑纹变化，只暗示不暴露）：
  1. 林婉儿低头看右手
  2. 手背上的剑纹温度在撤离过程中达到最高点，此刻正在慢慢消退
  3. 但剑纹的颜色——原本只是皮肤下的淡白色印记——现在变成了极浅的银灰色
  4. 像是吸收了遗迹中的某种能量
  5. 她将手缩入袖中，没有让李长风看到
- 过渡方式/结尾：
  1. 李长风站在石屋前，望着被封住的裂缝
  2. 手中握着那几页残破的笔记
  3. 他低头翻看，其中一页上有熟悉的字迹——与铜牌上的楷书一致
  4. 写着："归元宗内门，云崖子，瘴眼之下——"
  5. 后面的字被污渍覆盖，无法辨认
  6. 远处，遗迹外围方向传来异兽的低吼声，比之前更远了

## 写作风格要求
- 转折冲击：遗迹崩塌的紧迫感，节奏加快但不要慌乱
- 动作描写：撤离过程要紧凑，重点在两人配合，不是炫技
- 环境描写：崩塌的紧迫感——碎石、裂缝、入口封闭
- 节奏控制：快节奏撤离→慢节奏回到地面后的发现
- 对话：极简，危机时刻不说废话
- 结尾钩子：信息悬念（笔记残页）+状态异常（剑纹变色）
