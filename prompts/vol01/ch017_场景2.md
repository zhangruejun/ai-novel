## 场景二：地下大厅
- 地点：地下大厅（遗迹核心区域），约三丈见方
- 时间：Day 6 上午→中午
- 出场角色：李长风、林婉儿
- 大厅环境（简洁描写，不要堆砌）：
  1. 石阶尽头是一个约三丈见方的地下大厅
  2. 顶部有裂隙，少量天光从上方瘴气中透下来，形成几道灰绿色的光柱
  3. 大厅地面铺着完整的青石板
  4. 中央有一个已经干涸的圆形水渍痕迹——可能是某种阵法或水池
  5. 四周墙壁上有四个壁龛，其中三个空着，一个里面有物品
- 残缺阵法：
  1. 大厅地面上有明显的阵法痕迹——以中央水渍为中心，向外延伸出九条浅沟，呈放射状排列
  2. 每条浅沟的末端有一个小凹槽
  3. 阵法的大部分已经被岁月腐蚀得模糊不清，但基本结构还在
  4. 李长风辨认出这是一个"锁灵阵"——与石屋中的"聚灵阵"配合使用，将灵气锁定在特定区域
  5. 阵法已经失效，但残留的阵纹还在微微发光
- 云崖子遗骸（描写要克制，带敬意，不是恐怖场景）：
  1. 在壁龛下方的地面上，靠墙坐着一具遗骸
  2. 遗骸穿着残破的道袍，道袍的领口和袖口有云纹刺绣——与铜牌上的云纹一致
  3. 遗骸保持坐姿，背靠墙壁
  4. 左手握着一柄剑（剑身已锈蚀大半）
  5. 右手搭在膝上，指骨间夹着一枚玉简
  6. 遗骸的头微微低垂，像是在沉思，又像是在等待
- 遗物检查：
  1. 李长风蹲下身，检查遗骸和周围
  2. 遗骸旁边有一个破损的储物袋（已风化，内容物散落）
  3. 里面有几块碎裂的灵石（灵气耗尽，变成普通石头）、两枚失效的符箓、一本残破的笔记（纸张脆化，一碰就碎）
  4. 李长风小心地拿起玉简
- 玉简信息（信息悬念，不要过度渲染）：
  1. 玉简入手冰凉，表面有细微的裂纹
  2. 李长风用真气试探——玉简有反应，微微发亮
  3. 他将玉简贴在额头，一股信息流入脑海
  4. 信息内容："瘴非天降，乃人所为。归元宗内门弟子云崖子，奉命封印瘴眼，同伴尽殁，唯余一人。瘴眼之下，有物未明，不可轻动。若后来者见此简，切勿——"
  5. 信息到此中断，像是被什么力量强行切断
- 机关触发：
  1. 李长风取下玉简的瞬间，大厅地面微微震动
  2. 中央干涸的水渍痕迹中，九条浅沟同时亮起微弱的灰绿色光芒——不是灵气，而是瘴气
  3. 阵法被触发了，但不是原来的锁灵阵，而是某种被瘴气污染的变异阵法
  4. 灰绿色的瘴气从浅沟中涌出，向四周扩散
- 小冲突（不是战斗，是诡异场景）：
  1. 瘴气从地面涌出，形成低矮的雾气层，约到脚踝高度
  2. 雾气中开始出现某种东西的轮廓——不是异兽，而是阵法残留的"影像"
  3. 几个模糊的人形在瘴气中行走，动作僵硬，像是被什么力量操控
  4. 李长风拉着林婉儿后退，避开瘴气覆盖区域
  5. 影像没有攻击性，只是在原地重复某种动作——像是当年的封印场景被阵法记录下来
- 过渡方式：李长风将铜牌、玉简和能带走的物品收入行囊，准备撤离。但大厅顶部开始出现裂缝，碎石落下——遗迹开始不稳定

## 写作风格要求
- 悬疑氛围：通过遗骸、遗物、玉简信息推进线索，不要直接告诉读者全部答案
- 动作描写：李长风检查遗物、取玉简的动作要具体，不要写"一股暖流"等AI高频词
- 阵法残留的诡异感：灰绿色瘴气形成的影像，重复当年的场景，营造诡异但不恐怖的氛围
- 节奏控制：先慢后快——探索遗骸时慢节奏，机关触发后节奏加快
- 对话：简短，配合紧张的探索氛围
