# 第005章：断剑认主

断剑在指尖微微震颤。

李长风的手停在半空中，感受到剑柄传来的那一丝温热震动。不是错觉。那把沉寂了不知多少岁月的断剑，像是被什么东西唤醒了一样，发出微弱的嗡鸣声。

他没有急着握住它。

嗡鸣声越来越强，断剑表面的锈迹似乎在微微颤动，那些附在剑身上的灰尘也开始缓慢飘浮起来，像是被一股看不见的力量搅动了。

李长风蹲在断剑旁边，没有急着伸手去拿。

游戏里，获得这把剑只需要点击"拾取"按钮，然后它会自动进入背包。

他试着集中注意力，在心里默念了一声"拾取"。

……没有任何反应。

断剑还是安安静静地躺在那里。

他又试了一次，这次更明确地在脑海中想象着打开背包、点击"拾取"的操作流程。

还是没有反应。

"看来不行。"他喃喃自语。

游戏里的拾取机制在这个真实世界里不起作用。这也说得通——游戏里的拾取是系统帮你完成一系列隐藏操作，但这个世界有系统吗？至少目前为止，系统只给了背包和属性面板，并没有表现出能帮他"拾取"物品的功能。

李长风伸出手，直接握住了断剑的剑柄。

就在指尖触碰到剑柄的那一瞬间——

断剑猛地一震。

那股震颤比之前强烈了十倍不止，震得李长风的手差点松脱。他下意识地握紧了剑柄，但没有松开。

剑柄传来的不是排斥感，而是一种……渴求。像是这把剑在等待什么东西已经等了太久。

李长风仔细感受着那股震颤。

游戏里的装备拾取后会自动绑定，不需要额外操作。但这把断剑的震颤说明它在"等待"——等待某种激活条件。

他的脑海中闪过一个念头。

游戏里有个背景设定：上古时期的武器认主，需要用使用者的血液激活剑内刻印的符文。当然，在游戏里这只是剧情背景，玩家不需要真的这么做——点一下鼠标，系统就会自动播放一段"滴血认主"的动画效果。

但如果这个世界是真实的……

他低头仔细看剑柄。腐朽的木质纹理中，隐约能看到一些细密的纹路——不像是天然木纹，更像是某种人为雕刻的图案。他凑近了一些，用手指沿着纹路轻轻抚摸。

那些纹路组成了一个图案。

符文。

血液含有一个人的生命信息。如果这些符文真的存在，血液可能是激活它们的钥匙。

这不是凭空猜测。原主的记忆碎片中也有模糊的片段——村里的老人讲过，上古时期的高手炼器，会在武器内部刻入符文，只有用使用者的血才能激活。

原主不懂修炼，但那些零碎的记忆至少告诉他一件事：这个世界的武器认主，不是点击鼠标就能完成的。

值得一试。

最坏的结果也不过是什么都不发生。

李长风收回手，用牙齿咬破右手食指指尖。一阵刺痛传来，鲜红的血珠从指尖渗出。他没有犹豫，将手指按在了断剑的剑柄上。

血珠渗入剑柄腐朽的木质纹理中。

起初没有任何反应。

然后——

断剑猛地一震。

那股震颤比之前强烈了十倍不止，震得李长风的手差点松脱。他下意识地握紧了剑柄，接着感觉到一股灼热从剑柄上传来，顺着手指的经脉一路钻进体内。

"嗡——"

断剑发出一声低沉的鸣响，声音在狭小的石室中回荡。紧接着，剑身上那些密布的锈迹开始一点一点剥落，像是被什么力量从内部推了出来。

一股淡白色的微光从剑身中散发出来。

光芒很柔和，不刺眼。微光所到之处，灰尘纷纷散去，原本暗淡的剑刃开始露出原本的金属光泽——不是锈迹斑斑的废铁，而是经过千锤百炼的精钢。

李长风屏住呼吸。

他能感觉到，断剑正在吸收什么东西。

不是吸收外界的能量，而是吸收……他体内的东西。

一股细微的吸力从剑柄传来，沿着手指的经脉一路延伸到体内。那股本该让人不适的吸力却带着一种奇异的温和感，像是有人在帮他梳理体内淤塞已久的真气。

原主在这具身体里修炼了那么久，经脉中积累了大量的杂质和废真气。这些东西就像是下水管道里的污垢，堵住了真气的正常流动。而断剑的吸力，正在一点一点将这些杂质抽出来，经过某种转化之后吞噬掉。

淡白色的微光越来越亮，李长风能清晰地看到，那些被抽出的杂质在离开身体的瞬间被断剑吸收，然后转化为一种更加精纯的能量，又反哺回他的体内。

循环往复。

这个过程持续了大约半柱香的时间。

当最后一丝杂质被抽离的时候，断剑的淡白色微光缓缓收敛，最终完全消失。石室重新归于安静，只有岩壁上萤石的蓝色微光还在闪烁。

李长风缓缓睁开眼睛。

他感觉……不一样了。

之前那无处不在的疼痛消失了。肋骨不再像断了一样疼，胸口不再像压了一块石头，经脉中那股干涸的真气不知道什么时候变成了一条欢快流淌的小溪，在体内自由流动。

他低头看了看自己的身体。之前衣服上沾染的血迹已经干了，但伤口……已经结痂。不是那种刚结痂的脆弱状态，而是已经愈合了大半，只剩下淡淡的红印。

"伤势痊愈了。"

他活动了一下双臂，握了握拳头。力量感回来了。不，不是回来——是比原来更强了。

李长风闭上眼睛，感受体内的真气。

之前他的真气总量大约是10/100，微弱到几乎感觉不到。现在——他试着运转了一个周天，真气从丹田出发，流经四肢百骸，每一处经脉都畅通无阻。真气的流动前所未有的顺畅。

剑徒境初期……不。

他能感觉到，体内的真气量已经远远超出了初期的范畴。经脉的宽度也变大了，真气池的容量也随之扩展。

剑徒境巅峰。

从初期直接跳到了巅峰，中间没有经历中期和后期。这是因为断剑帮他疏通了所有经脉，扫清了所有障碍。杂质被清除之后，他的真气积累已经足够冲击下一个大境界，只是暂时还没有突破的契机。

但这已经够了。

从一个重伤濒死的剑徒境初期，到一个全盛状态的剑徒境巅峰。

只用了一把断剑。

"爽。"李长风从牙缝里挤出一个字。

他低下头，再次看向手中的断剑。

剑身上的锈迹已经完全剥落了。露出来的剑刃虽然只有两尺长的残段，但刃口锋利得让人心寒。他用拇指轻轻刮了一下刃口，指腹立刻传来一道细微的血线——这把剑的锋利程度，远超他现在这个境界应有的水平。

剑柄上那个古篆字"云"也清晰可见。

李长风凑近端详了一番。笔画古拙，每一笔都像是用刻刀深深凿进去的。他试着在脑海中唤出系统面板——

【检测到未知文字……无法识别】

还是无法识别。

李长风收回目光，不再纠结于此。这个伏笔以后再说，现在他手里有了武器，实力暴涨，接下来的计划可以推进了。

他试着挥舞了两下断剑。

"呼——呼——"

断剑在空气中划出两道短促的破空声。重量适中，重心靠前，挥剑的时候有一种自然的惯性，像是这把剑本来就应该属于他一样。

李长风将断剑收入腰间——他没有剑鞘，只能暂时这样带着。然后他看了看石室角落里的那把系统赠送的新手短剑，插在墙缝里，留在这里。

断剑比新手短剑强了不止一个档次，新手短剑已经没有携带的必要了。

他最后扫了一眼石室，确认没有遗漏任何东西，然后转身走出石室。

矿洞深处的黑暗依然在前方等着他。

但这一次，他不再是一个手无寸铁的濒死穿越者。

他有剑了。

李长风沿着来时的路往回走了一段，然后在三岔口转了一个方向——不是回头的路，而是朝矿洞更深处走去。

游戏先知的地图上，一个红色的BOSS标记正在远处闪烁。

石皮野猪。

黑风矿洞深处的精英BOSS，等级5级，对应剑徒境巅峰到剑师境初期的实力。在游戏中，这个BOSS需要一个小队才能击杀。但现在——

他知道它的弱点。

左眼下方三寸，是石皮野猪唯一的破绽。那个位置的甲壳最薄，只要能命中，一剑就能穿透。

"先去拿第一桶金。"

李长风握了握手中的断剑，脚步越来越快。

矿洞深处的妖气，已经隐隐能够感受到了。
