# 第018章 场景3：瘴气异变

两人刚走出岩洞不到十步，地面传来轻微震动。

不是遗迹崩塌那种剧烈的晃动，而是从地底深处传来的持续微颤。脚底能感觉到一层层细密的颤动顺着鞋底传上来，频率不快，但没有停。

李长风停下脚步，蹲下身，手掌贴在地面上。震源在下方极深处，方向与遗迹位置一致。

他站起身时，周围的瘴气已经开始变化。

原本已经变淡的浅灰色瘴气突然变得浓稠，颜色从浅灰转为暗灰。隐约有一丝绿色在灰气中流动，像水面上浮着的油纹。瘴气流动速度加快，不再贴着地面缓慢爬行，而是像被什么力量搅动，地面上浮现出数个小型漩涡，边缘翻卷，中心下陷。

漩涡越转越快，最终合为一股，从地底直冲而起，形成一根粗壮的瘴气柱。灰绿色的气柱拔地而起，高度远超岩洞，直往天空冲去。

李长风一把拉住林婉儿往回退。

"遗迹封闭后，瘴气源头失去了封印的压制。"他说。

之前的变淡只是短暂的能量真空期。封印彻底失效了。

瘴气浓度在快速上升，已经超过了进入遗迹前的水平。李长风从行囊中取出解毒丹，递给林婉儿一枚，自己服下一枚。丹药入腹，胸口升起一股温热，但很快被周围瘴气的侵蚀感抵消。

"不能停，往高处走。"

两人向瘴谷外围的高地撤离。李长风步子迈得快，右腿伤口在快速移动中再次被扯开，血渗出新的布条，在裤管上洇开一片深色。他步子微了一下，没有停。

林婉儿跟在身后，右手始终缩在袖中。手背上的银灰色剑纹在瘴气异变后再次开始发热，温度比在遗迹中更高，但没有烫到无法忍受。她将右手紧紧攥成拳，指甲陷入掌心。

撤离到一处坡地时，瘴气中浮现出一个轮廓。

一只异兽从灰绿色的雾中走出来。不是之前在瘴谷中遇到的那种——这只体型更大，约莫牛犊大小。外形像放大的蜥蜴，背部排列着一排骨刺，每根有手指长短，尖端发黑。眼睛是浑浊的黄色，瞳孔竖成一条细线。

它盯着两人看了三息。没有立刻攻击。

李长风拔出无名铁剑，横在身前。剑刃上的暗绿斑点还在，握在手里分量熟悉。他没有主动进攻——右腿伤势不允许他进行大幅度移动，强行出手只会露出破绽。

异兽也没有进攻。它站在原地，黄色的眼睛盯着两人，头微微偏了偏，像是在辨认什么。然后它缓缓后退，一步，两步，身体边缘重新被瘴气吞没，消失不见。

李长风看着它退走的方向。异兽的退让不是畏惧，更像是……在观察。

两人继续撤离，攀上一处高地。

回头望去，瘴谷方向的瘴气已经浓到看不见地面。灰绿色的瘴气形成一根粗壮的柱子，从地底直冲天空。柱子顶端有某种东西在闪烁——不是光，而是某种规则的、有节奏的脉动，像是心跳。

李长风盯着那根瘴气柱，右手无意识地握住了胸前的玉简。玉简表面的裂纹中，灰绿色的光芒随着瘴气柱的脉动频率一亮一暗。

"瘴眼……开了。"他低声说。

林婉儿站在他身后，手背上的银灰色剑纹烫得几乎要烧穿皮肤。

她将右手紧紧攥成拳，指甲陷入掌心。
