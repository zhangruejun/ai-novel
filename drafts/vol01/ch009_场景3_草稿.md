# 第009章 场景3 草稿：决意

天快亮的时候，李长风做了一个决定。

他握住无名铁剑。不是拿起来，是握住。五根手指扣紧剑柄，掌心贴着麻绳缠绕的地方。绳已经磨得快断了，但他没有松开。

真气从掌心的点流出，进入剑身。

剑刃的缺口还在，剑脊的划痕也还在。但剑不再发沉。之前挥剑时那种重心偏移的感觉消失了。剑身与手臂连成了一条线，从肩到腕，到掌，到剑尖。

合手。

门外传来脚步声。缓慢，沉稳，是老人的步子。

门被推开。赵老汉站在门口，衣服上沾着灰，袖口的裂口还没缝。他看到李长风握着剑的样子，停住了脚步。

"后生……"老汉的目光落在那把缺口累累的铁剑上，停了半晌，"这剑……不合适了。"

李长风没有回答。

他低头看剑。剑刃上全是缺口，大的半寸，小的如米粒。但在晨光中，剑身泛着一种光。不是锋刃的冷光，是铁被磨过之后留下的那种暗光。沉稳，不刺眼。

老汉等了一会儿，见他不说话，又道："村里有铁匠。我去打一把新的。"

"不用了。"

李长风举起剑，对着门外的光看。

晨光从门外照进来，穿过剑身。剑刃的缺口在阳光下清晰可见，像是一张残缺的地图。但剑的整体线条是流畅的。从剑柄到剑尖，一气呵成。重心就在掌心前方三寸，正好是手腕发力的最佳位置。

"这把就行。"

赵老汉看着他。嘴唇动了一下，像是要说什么，最终没有说出口。老汉点了点头，转身走了。脚步声渐渐远去。

李长风把剑挂回腰间。麻绳绕了两圈，打了个结。剑坠在腰侧，分量刚好。

叮——

一声脆响在脑海中响起。

面板浮现。字迹清晰。

【无名铁剑已认主】
【剑道天赋：初显（未入门→剑意初悟）】
【真气觉醒：完成（引气入体正式完成）】

面板停留三息，消散。

李长风推开门，走出茅屋。

晨雾正在散去。远处的山脊露出轮廓，层峦叠嶂，像一排剑峰。他抬头看向黑风寨的方向，手搭在腰间的剑柄上。

麻绳粗糙，剑柄温热。

路在脚下。
