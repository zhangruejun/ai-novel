# AI辅助写作控制指南

## 一、AI写网的典型问题与解决方案

### 1.1 常见AI写作问题

| 问题 | 表现 | 原因 |
|------|------|------|
| **逻辑崩坏** | 前后矛盾、人物行为不符合设定、力量体系混乱 | AI缺乏长期记忆，无法记住前文设定 |
| **节奏失控** | 一章写三天的事、一章跨越半年 | AI没有章节节奏感概念 |
| **模板化严重** | 每章结构相似、对话套路化、战斗描写千篇一律 | AI倾向于使用最常见的模式 |
| **细节丢失** | 忘记前文伏笔、角色名字写错、物品消失 | AI上下文窗口有限 |
| **情感空洞** | 感情发展突兀、角色反应不符合性格 | AI缺乏真实情感体验 |
| **AI痕迹明显** | 过度使用"然而""与此同时""值得注意的是"等连接词 | AI训练数据的语言习惯 |

### 1.2 核心解决思路

**不要让AI"自由创作"，而是让AI"填空执行"**

## 三、卷级规划模板

每卷开始前，制作以下规划文档：

### 3.1 卷概要

```markdown
# 第X卷：[卷名]

## 基本信息
- 章节范围：第XXX-XXX章（共XX章）
- 核心剧情目标：[一句话概括本卷主要剧情]
- 主角境界变化：[起点] → [终点]
- 关键角色状态变化：
  - 主角：[起点状态] → [终点状态]
  - 女主：[起点状态] → [终点状态]
  - 配角A：[起点状态] → [终点状态]

## 关键节点（按章节顺序）
| 章节 | 核心事件 | 情绪基调 | 结尾钩子 |
|------|----------|----------|----------|
| XXX | [事件描述] | 轻松/紧张/悲壮 | [钩子内容] |
| XXX | [事件描述] | 轻松/紧张/悲壮 | [钩子内容] |

## 伏笔设计
- 新埋设的伏笔：[描述]
- 需要回收的伏笔：[描述]
- 伏笔回收章节：[章节号]

## 反派行动线
- 反派A在本卷的行动：[描述]
- 反派B在本卷的行动：[描述]

## 注意事项
- [特殊要求1]
- [特殊要求2]
```

## 五、Prompt模板设计

### 5.1 基础Prompt模板

```
请根据以下要求，写出第XXX章的内容：

## 章节信息
- 章名：[XXX]
- 核心事件：[一句话概括]
- 情绪基调：[轻松/紧张/悬疑/热血/悲壮]
- 字数要求：约3000字

## 当前角色状态
- 主角李长风：[当前境界][当前状态，如"左臂有伤""心情沉重"]
- 女主林婉儿：[当前境界][当前状态][与主角关系阶段]
- [其他出场角色及状态]

## 场景一：[场景名]
- 地点：[具体地点]
- 时间：[具体时间]
- 出场角色：[列出]
- 核心动作：[描述]
- 对话要点：[描述对话的核心内容或冲突]

## 场景二：[场景名]
- 地点：
- 时间：
- 出场角色：
- 核心动作：
- 对话要点：

## 场景三：[场景名]
- 地点：
- 时间：
- 出场角色：
- 核心动作：
- 对话要点：

## 结尾要求
- 钩子类型：[悬念/危机/反转]
- 具体要求：[描述]

## 写作风格要求
- 战斗描写：突出剑道特色，强调策略而非数值，有动作细节
- 对话风格：符合角色性格，李长风冷静果断，林婉儿聪明缜密
- 环境描写：简洁有画面感，不要过度堆砌辞藻
- 节奏控制：[快节奏/慢节奏/张弛有度]

## 禁止事项
- 不要使用"然而""与此同时""值得注意的是""不得不提"等AI常用连接词
- 不要让角色做出不符合性格的行为
- 不要忽略角色的当前状态（如有伤就是有伤）
- 不要添加大纲中没有的剧情
- 不要出现现代用语（如"OK""没问题"等）
- 不要使用"他感到一阵复杂的情绪涌上心头"等空洞的心理描写
```

### 5.2 战斗场景专用Prompt

```
## 战斗场景特殊要求
- 对手信息：[对手名称][实力][武器][特点][弱点]
- 战斗阶段：
  1. 试探阶段（约XX字）：双方交手，互相试探
  2. 交锋阶段（约XX字）：展示对手实力，主角陷入劣势
  3. 分析阶段（约XX字）：主角观察对手，发现弱点
  4. 反击阶段（约XX字）：主角利用弱点，扭转局势
  5. 决胜阶段（约XX字）：主角施展绝技，击败对手
- 战斗特色：[如"剑意对拼""身法较量""阵法困战"]
- 战斗代价：[主角是否受伤/消耗/获得感悟]
- 旁观者反应：[如有旁观者，描写他们的反应]
```

### 5.3 感情场景专用Prompt

```
## 感情场景特殊要求
- 感情阶段：[初识/暧昧/确认关系/生死与共]
- 场景氛围：[温馨/紧张/悲伤]
- 互动要点：
  1. [如"两人因某事产生分歧"]
  2. [如"对话中暗含关心"]
  3. [如"一个细节体现默契"]
- 不要直白写"我爱你""我喜欢你"等，用行为和细节体现感情
```

## 七、一致性检查清单

### 7.1 每章完成后检查

```markdown
## 一致性检查清单

### 角色一致性
- [ ] 角色名字是否正确（李长风、林婉儿、张云义等）
- [ ] 角色当前境界是否正确
- [ ] 角色当前状态是否正确（是否有伤、是否在附近等）
- [ ] 角色性格表现是否符合设定
- [ ] 角色称呼是否一致（如张云义叫主角"大哥"等）

### 剧情一致性
- [ ] 时间线是否正确（是否跳跃/倒流）
- [ ] 地点是否正确（角色是否在合理的位置）
- [ ] 前文提到的物品/信息是否被正确使用
- [ ] 伏笔是否被正确处理（埋设/回收）
- [ ] 反派行动是否符合其动机

### 力量体系一致性
- [ ] 境界等级是否正确（不要出现大纲中没有的境界）
- [ ] 战斗结果是否合理（越级战斗是否有合理解释）
- [ ] 技能使用是否符合设定

### 语言质量
- [ ] 是否有禁用AI词汇
- [ ] 是否有现代用语（应使用古风用语）
- [ ] 对话是否符合角色性格
- [ ] 是否有重复的句式或描写
- [ ] 是否有过度空洞的心理描写

### 节奏检查
- [ ] 本章是否有明确的核心事件
- [ ] 场景转换是否自然
- [ ] 结尾是否有钩子
- [ ] 字数是否在2800-3200字范围内
```

### 7.2 每10章进行一次深度检查

```markdown
## 深度一致性检查（每10章一次）

### 剧情进度
- [ ] 当前进度是否符合卷级规划
- [ ] 关键节点是否按时完成
- [ ] 是否有偏离大纲的剧情

### 角色成长
- [ ] 主角成长是否符合预期
- [ ] 配角戏份是否合理
- [ ] 感情发展节奏是否正确

### 伏笔追踪
- [ ] 已埋设的伏笔列表：[列出所有未回收伏笔]
- [ ] 计划回收的伏笔：[列出计划在哪些章节回收]
- [ ] 是否有遗漏需要回收的伏笔

### 反派进度
- [ ] 各反派当前状态
- [ ] 反派是否有合理戏份
- [ ] 反派行动是否符合其逻辑

### 读者体验
- [ ] 连续3章是否有相似结构（模板化检测）
- [ ] 是否连续5章没有高潮（节奏检测）
- [ ] 是否有让读者"出戏"的内容
```

## 十、分场景生成策略

### 10.1 为什么要分场景生成？

AI在一次生成中超过1500字后，质量会明显下降。建议将一章（3000字）拆分为2-4个场景，分别生成后拼接。

### 10.2 分场景生成示例

```
第XXX章分为3个场景生成：

场景一生成Prompt：

场景二生成Prompt：
→ 输入场景二的细纲
→ 附带场景一的结尾内容（让AI知道前情）
→ 要求生成约1000字
→ 检查场景二质量

场景三生成Prompt：
→ 输入场景三的细纲
→ 附带场景二的结尾内容
→ 要求生成约1000字
→ 检查场景三质量

→ 手动拼接三个场景
→ 润色过渡段落
```

## 十一、质量评估标准

### 11.1 每章质量评分

```
评分维度（每项10分，满分100分）：

1. 剧情推进（10分）：本章是否推进了剧情？
2. 角色塑造（10分）：角色行为是否符合性格？
3. 对话质量（10分）：对话是否自然？是否符合角色？
4. 战斗描写（10分）：战斗是否有策略性？是否有细节？
5. 情感表达（10分）：感情是否通过行为和细节体现？
6. 节奏控制（10分）：节奏是否符合预期？
7. 结尾钩子（10分）：结尾是否有吸引力？
8. 语言质量（10分）：是否有AI痕迹？是否流畅？
9. 一致性（10分）：是否与前文一致？
10. 创新度（10分）：是否有新鲜感？是否避免模板化？

得分标准：
- 90-100分：优秀，可以直接使用
- 80-89分：良好，微调后使用
- 70-79分：一般，需要较多修改
- 60-69分：较差，建议重写
- 60分以下：不合格，必须重写
```

## 十二、常见问题Q&A

### Q1: AI写的对话总是很生硬怎么办？

**A**: 在Prompt中加入对话示例：
```
对话风格参考：
"好剑。"李长风手指轻抚剑身，剑鸣如龙吟。
林婉儿微微一笑："配你正好。"
——注意：对话简短，用行动和细节补充对话之外的信息。
```

### Q2: AI写的战斗总是千篇一律怎么办？

**A**: 每场战斗在Prompt中指定不同的战斗特色：
```
本场战斗特色：
- 不是力量对拼，而是身法较量
- 重点描写脚步移动和闪避
- 主角通过观察对手的移动规律发现破绽
- 最终一击是预判对手落点，提前出剑
```

### Q3: 如何防止AI忽略前文设定？

**A**: 使用"角色状态卡"和"关键信息"附在每次Prompt前：
```
【重要提醒：以下信息必须遵守】
- 主角左臂有伤，出剑速度受影响
- 林婉儿不会武功，但在场
- 当前场景在下雨天
- 主角的剑有裂缝，不能全力施展
```

### Q4: AI写的内容总是很平淡怎么办？

**A**: 在Prompt中加入"情绪锚点"：
```
本章情绪锚点：
- 场景一：压抑（被强敌追杀，无处可逃）
- 场景二：转折（发现隐藏洞穴，一线生机）
- 场景三：希望（获得传承，实力大涨）
→ 每个场景必须让读者感受到对应的情绪
```

### Q5: 如何检测AI痕迹？

**A**: 重点检查以下特征：
1. 过度使用连接词（然而、与此同时、值得注意的是）
2. 空洞的心理描写（内心久久不能平静）
3. 套话式环境描写（月光如水、微风拂面）
4. 套路化战斗（你来我往、难解难分）
5. 不自然的过渡（且说、话说）
6. 对称句式（他做了A，也做了B；既有C，又有D）

## 十三点五、实战技巧与避坑指南

### 13.5.1 Prompt编写技巧

#### 1. "正向指令"优于"负面禁止"

❌ 差写法：
```
不要写得太平淡
不要写得很无聊
```

✅ 好写法：
```
场景一开始就要有冲突：XXX闯入了XXX的闭关之地，被守卫发现
对话要简短有力，每句不超过20字
```

**原因**：AI对"不要"的指令理解较弱，它更容易执行"做什么"而不是"不做什么"。负面指令放在禁用语列表中作为最后把关即可。

#### 2. 用"示例"代替"描述"

❌ 差写法：
```
对话要简短有力，符合武侠风格
```

✅ 好写法：
```
对话风格示例：
"拔剑。"
"你不配。"
"那就试试。"
——短句为主，避免长篇大论的解释
```

#### 3. 用"具体数据"代替"模糊要求"

❌ 差写法：
```
写一场精彩的战斗
```

✅ 好写法：
```
战斗要求：
- 共3个回合，每回合约300字
- 第1回合：对手进攻，主角防守，展示对手实力
- 第2回合：主角试探，发现对手左腿有旧伤
- 第3回合：主角专攻左腿，3剑之内破敌
- 主角使用XXX剑法的前三式
- 对手使用XXX刀法
```

#### 4. 控制"创造性空间"

- **需要AI发挥时**（如环境描写、配角对话）：给方向，不给细节
- **需要AI精确时**（如剧情关键点、伏笔回收）：给足细节，限制发挥

### 13.5.2 常见陷阱与解决方案

#### 陷阱1：AI会"忘记"你已经写过的内容

**表现**：你让AI写第100章，它不知道第99章结尾发生了什么。

**解决方案**：
- 每次Prompt必须附带"前章结尾"（最后200-300字）
- 在Prompt开头加一段"当前状态摘要"
- 使用类似这样的格式：
```
【前情提要】
上一章结尾：李长风收剑入鞘，转身看向洞穴深处。林婉儿拉住他的衣袖："里面有东西在动。"
当前状态：李长风（真气剩余六成，左臂有伤），林婉儿（无战斗力，持解毒丹x2）
```

#### 陷阱2：AI会"注水"

**表现**：你要求写1000字，AI写了大量无意义的描写、重复的心理活动、冗长的过渡。

**解决方案**：
- 明确字数范围（如"800-1000字"），不要只说"约1000字"
- 在Prompt中指定每个场景的"内容配额"：
```
内容配额要求：
- 对话占40%（约400字）
- 动作描写占30%（约300字）
- 环境/心理等其他占30%（约300字）
```
- 生成后如果发现注水严重，**果断删掉重写**，不要试图修补

#### 陷阱3：AI会"过度发挥"

**表现**：你只让它写一场战斗，它顺便把战后总结、感情发展、新角色登场全写了。

**解决方案**：
- 在Prompt末尾加上明确的范围限制：
```
【范围限制】
- 只写到XXX为止
- 不要写战后的内容
- 不要添加大纲中没有的角色
- 不要推进感情线
```

#### 陷阱4：AI的"中庸之道"

**表现**：AI倾向于写"平均"的内容，不会写出特别好的也不会写出特别差的。高潮戏不够燃，悲伤戏不够虐。

**解决方案**：
- 使用"极端化指令"：
```
本场战斗要让读者感到"绝望"：
- 主角完全被压制，毫无还手之力
- 同伴受伤倒地，无力支援
- 对手的压倒性优势要体现在每一个细节上
```
- 用"情绪锚点"强化氛围（见Q4）

#### 陷阱5：AI的"角色同质化"

**表现**：所有角色说话风格相似，行为模式相似，仿佛换了同一张脸。

**解决方案**：
- 为每个重要角色建立"角色卡"，每次Prompt附带：
```
【角色卡·林婉儿】
- 性格：聪慧、冷静、外冷内热
- 说话风格：简短、直接、偶尔带刺
- 口头禅/习惯：不说废话，不称呼亲昵称呼
- 对主角态度：表面客气，实际关心
- 示例对话：
  "你的剑法退步了。"
  "不是关心你，只是不想失去一个好打手。"
```

### 13.5.3 高效工作流建议

#### 1. 建立"素材库"

在写作过程中，收集并整理以下内容：
- **好段落**：AI偶尔会写出很棒的句子，收集起来作为风格参考
- **好对话**：收集角色之间的高质量对话，作为后续Prompt的示例
- **好战斗**：收集打得好的战斗描写，分析其结构和用词
- **AI高频错误**：记录AI常犯的错误，在后续Prompt中主动避免

#### 2. 建立"风格参考文档"

找2-3本你喜欢的网文/小说，从中摘录：
- 对话示例（展示你希望的对话风格）
- 战斗描写（展示你希望的战斗节奏）
- 环境描写（展示你希望的画面感）
- 感情描写（展示你希望的情感表达方式）

在每次Prompt中附上1-2段风格参考。

#### 3. 建立"常用Prompt片段库"

一些常用的Prompt片段，可以组合使用：
```
【节奏控制片段】
"本场景节奏要快，不要有冗长的描写。每2-3句对话就要有动作/冲突推进。"

【悬念制造片段】
"场景结尾要留下悬念：XXX发现了一封信，但只读了一半就被打断了。读者不知道信的内容。"

【情感升温片段】
"本场景要让两人关系更进一步，但不要直白表达感情。用一个细节体现：XXX注意到主角受伤，沉默地递上金疮药。"

【气氛压抑片段】
"本场景要让读者感到压抑：天气阴沉，周围安静得只能听见脚步声。角色对话简短，声音低沉。"
```

### 13.5.4 千章长篇管理技巧

#### 1. 建立"剧情进度表"

用一个表格记录每10章的剧情概要：

```
| 章节范围 | 核心事件 | 主角状态变化 | 重要伏笔 | 备注 |
|---------|---------|------------|---------|------|
| 1-10 | 穿越+入门 | 凡人→炼气三层 | 神秘玉佩 | 开篇 |
| 11-20 | 外门大比 | 炼气三层→六层 | 发现内门阴谋 | |
```

#### 2. 建立"角色出场记录"

记录每个角色出场的章节，便于追踪：

```
| 角色 | 出场章节 | 当前状态 | 备注 |
|------|---------|---------|------|
| 李长风 | 1（主角） | 炼气六层，左臂有伤 | |
| 林婉儿 | 3 | 炼气四层，持有解毒丹 | 好感度：好感 |
```

#### 3. 定期"回看"

- **每50章**：从头快速浏览一遍已写章节（可以只读每章摘要）
- **每100章**：做一次全面的一致性检查
- **换卷时**：重新审阅上一卷的内容

#### 4. "设定文档"维护

建立以下设定文档并持续更新：
- **世界观设定**：势力分布、地理、历史
- **力量体系**：境界等级、技能体系、战斗规则
- **物品设定**：重要物品、丹药、法宝
- **角色设定**：性格、背景、关系图

```markdown
- 推理过程要符合逻辑，不要凭空猜测
- 线索必须在前文有铺垫，不要突然出现
- 结尾要留下新的疑问或悬念
```

## 十六、AI写网的进阶策略

### 16.1 多模型协作策略

不同的AI模型有不同的优势，建议：

| 场景类型 | 推荐模型特点 | 原因 |
|---------|------------|------|
| 战斗场景 | 动作描写强、节奏感好的模型 | 战斗需要紧凑的节奏和具体的动作细节 |
| 感情场景 | 情感细腻、对话自然的模型 | 感情需要微妙的情绪表达和自然的对话 |
| 日常场景 | 对话流畅、幽默感强的模型 | 日常需要轻松自然的对话 |
| 悬疑场景 | 逻辑性强、善于埋伏笔的模型 | 悬疑需要严密的逻辑和巧妙的伏笔 |

**建议**：测试3-5个主流模型，为每个场景类型指定最适合的模型。

### 16.2 "迭代式生成"策略

不要指望一次生成就完美。使用"迭代式生成"：

1. **第一轮**：让AI生成初稿，关注剧情推进
2. **第二轮**：人工修改后，让AI基于修改版润色语言
3. **第三轮**：人工最终审校，调整细节

这种策略比"一次生成完美"更高效。

### 16.3 "风格学习"策略

让AI学习你的写作风格：

1. 选择你写的最满意的3-5章
2. 让AI分析这些章节的写作特点（对话长度、描写比例、用词偏好等）
3. 将分析结果整理成"风格指南"
4. 在每次Prompt中附上风格指南的关键点

### 16.4 "反向检查"策略

让AI自己检查问题：

```
请检查以下内容是否存在以下问题：
1. 是否有逻辑矛盾？
2. 是否有与前文不一致的地方？
3. 是否有过度使用AI常用词汇？
4. 是否有节奏问题（过快/过慢）？

如果有，请指出具体问题所在位置。
```

### 16.5 "分场景生成"策略优化

原文的10.2节提供了基础的分场景生成方法。以下是优化策略：

#### 1. 场景之间的衔接

- 生成场景N时，把场景N-1的**最后2-3段**附在Prompt中
- 要求AI在场景N开头**自然衔接**前文，不要重复前文内容
- 人工拼接时，检查过渡是否自然，必要时重写过渡段

#### 2. 场景长度控制

- **对话场景**：600-800字为宜（对话太多会空洞）
- **战斗场景**：800-1200字为宜（太短不够精彩，太长会疲劳）
- **过渡场景**：400-600字为宜（不要过长，快速推进剧情）
- **高潮场景**：1000-1500字为宜（需要足够的篇幅来营造氛围）

#### 3. 场景生成的顺序

对于复杂章节，建议按以下顺序生成：
1. 先生成**核心场景**（如战斗、关键对话）
2. 再生成**过渡场景**
3. 最后生成**开头和结尾**

这样能确保核心场景的质量不受影响。

## 十七、常见问题Q&A（补充）

### Q6: 写了100章之后，AI完全忘记了前面的设定怎么办？

**A**: 这是千章网文最大的挑战。解决方案：

1. **建立"设定百科"文档**：
   - 世界观、势力分布、力量体系
   - 所有角色档案（性格、背景、境界、关系）
   - 重要物品、伏笔、剧情线索
   - 每次开新卷时，把相关设定附在Prompt前面

2. **建立"剧情摘要"文档**：
   - 每10章写一段200字的剧情摘要
   - 开新章时，附上最近50章的剧情摘要
   - 这样AI就能"记住"最近发生了什么

3. **关键章节手动写**：
   - 涉及重要伏笔回收的章节，建议自己写
   - 高潮戏、结局戏，建议自己写或深度参与
   - AI主要负责过渡章节和常规战斗

4. **分卷独立管理**：
   - 每卷视为一个相对独立的故事
   - 卷与卷之间的设定可以略有调整
   - 这样即使AI忘记了早期设定，也不会影响当前卷的质量

### Q7: 如何保持1000章的新鲜感？

**A**: 1000章最大的敌人是"重复"。解决方案：

1. **剧情层面**：
   - 每卷引入新的核心冲突
   - 每50章有一次大的剧情转折
   - 不断引入新角色、新势力、新地图

2. **写法层面**：
   - 尝试不同的叙事视角（偶尔用配角视角）
   - 尝试不同的结构（偶尔用倒叙、插叙）
   - 尝试不同的节奏（偶尔有快节奏章节，偶尔有慢节奏章节）

3. **角色层面**：
   - 让角色不断成长变化
   - 让角色的关系不断变化
   - 给角色新的动机和目标

### Q8: AI写的章节没有"灵魂"怎么办？

**A**: 这很常见。AI擅长执行，不擅长创造"灵魂"。解决方案：

1. **在关键位置注入"灵魂"**：
   - 章节开头：用一个有冲击力的场景/对话开头
   - 章节结尾：用一个有感染力的钩子结尾
   - 这些关键位置建议自己写

2. **让AI写"骨架"，你来注入"血肉"**：
   - AI负责剧情推进、场景描写、基本对话
   - 你负责添加关键的细节、情感、金句

3. **建立"金句库"**：
   - 收集你写的好句子
   - 在合适的场景让AI参考
   - 或者自己手动插入

### Q9: 如何判断什么时候该用AI，什么时候该自己写？

**A**: 以下场景建议**自己写**：

- 开篇第一章（奠定全书基调）
- 重要伏笔的埋设（AI容易遗漏）
- 重要伏笔的回收（需要精确控制）
- 高潮戏/决战戏（需要情感和节奏的精准控制）
- 感情戏的关键节点（确认关系、生死离别等）
- 每卷的最后一章（需要强烈的钩子）

以下场景适合**用AI写**：

- 过渡章节（赶路、修炼、日常）
- 常规战斗（非关键战斗）
- 角色对话（有明确对话要点时）
- 环境描写
- 配角/配角的戏份

### Q10: AI写的东西总是"差一口气"怎么办？

**A**: 这是AI写作的常态。不要追求100分，追求80分即可：

1. **接受不完美**：AI写的东西达到80分就可以用了
2. **关键位置手动修补**：在关键对话、关键动作处手动修改
3. **把精力花在刀刃上**：1000章不可能章章精品，保证主线质量即可
4. **读者的容忍度比你想象的高**：只要剧情推进得当，读者不会在意细节瑕疵

### Q11: 如何管理每日写作进度？

**A**: 建议的每日写作节奏：

- **预热（15分钟）**：回顾前3章内容，确认角色状态和剧情进度
- **细纲（15分钟）**：制作今日要写的章节细纲
- **生成（30-60分钟）**：AI生成内容，分场景生成需要多次交互
- **审校（30-60分钟）**：人工审校，修改不合理内容，润色语言
- **记录（10分钟）**：更新角色状态卡、伏笔追踪表

总计约1.5-2.5小时/章。如果每天写2章，需要3-5小时。

## 十九、新手入门建议

### 19.1 前100章的学习曲线

对于刚开始用AI写作的新手，建议：

1. **第1-10章**：重点学习如何写有效的Prompt。这几章会很痛苦，质量可能不高，这是正常的。
2. **第11-30章**：逐渐找到感觉，知道怎么给AI指令了。质量开始提升。
3. **第31-50章**：建立了自己的素材库和风格参考。AI写作流程趋于稳定。
4. **第51-100章**：能比较高效地写作了。开始关注如何保持新鲜感和质量。

### 19.2 给新手的5条建议

1. **不要追求完美**：前50章的目标是"学会用AI"，不是"写出神作"。
2. **多存草稿**：重要的场景多生成几个版本，选最好的。
3. **及时记录错误**：AI犯的每个错误都记下来，下次在Prompt中提前防止。
4. **不要偷懒**：AI写得快不代表你可以不写细纲。细纲质量直接决定AI输出质量。
5. **坚持审校**：再忙也要人工审校。跳过审校=埋雷。

## 二十、总结

AI辅助写作的核心原则：

1. **AI是执行者，不是创作者** —— 你提供所有创意和结构，AI负责填充文字
2. **细节决定成败** —— 给AI的指令越详细，输出质量越高
3. **人工审校不可少** —— AI生成的内容必须经过人工检查和修改
4. **建立系统化管理** —— 角色状态、伏笔追踪、剧情进度都要记录在案
5. **持续优化** —— 根据AI的表现不断调整Prompt和流程
6. **保持个人风格** —— 人工润色环节是注入个人风格的关键步骤
7. **关键内容自己写** —— 开篇、高潮、结局等关键位置建议亲自操刀
8. **建立素材库** —— 收集好段落、好对话、好战斗描写，作为风格参考
9. **管理长期记忆** —— 用设定百科和剧情摘要弥补AI的记忆缺陷
10. **保持耐心和信心** —— AI写作是一场马拉松，前100章可能很痛苦，但之后会越来越顺

**最后的话**：这份指南提供了一套系统化的AI辅助写作方法。但它不是死板的规则，而是可以参考的框架。随着你的经验积累，你会发现哪些方法适合自己，哪些需要调整。重要的是**开始行动**，在实践中不断优化自己的写作流程。

祝你写作顺利！
