# 任务启动检查清单

> **用途**：每次AI收到写作任务时，必须先执行此检查，验证不通过则拒绝执行。
> **使用方式**：复制到AI对话中，或作为SOP第0.5节的执行辅助工具。
> **版本**：v1.0
> **创建日期**：2026-04-10

---

## 任务启动验证流程（铁律·防跳过）

### 第一步：检查任务指令格式

收到任务指令后，逐项检查以下要素：

| 序号 | 检查项 | 是否存在 | 说明 |
|------|--------|---------|------|
| 1 | 项目名称 | [ ] | 如"剑道通神" |
| 2 | 当前工作目录 | [ ] | 如"e:/Novel/ai-novel" |
| 3 | 目标章节 | [ ] | 如"第6章 至 第10章" |
| 4 | 每章目标字数 | [ ] | 如"2800-3200字" |
| 5 | 必读文档列表 | [ ] | 至少包含SOP、角色状态卡、伏笔追踪表 |
| 6 | 执行规则 | [ ] | 一次只写一章、顺序执行、每章8步 |

### 第二步：判定结果

**全部勾选（6/6）**：
- ✅ 任务指令符合SOP规范
- 回复："已确认任务指令符合SOP规范，开始执行写作任务。"
- 进入SOP第1步：准备工作

**未全部勾选（<6/6）**：
- ❌ 任务指令不符合SOP规范
- 回复："⚠️ 任务指令不符合SOP规范，无法执行。请按以下格式重新发送任务指令：[引用SOP第0.1节模板]"
- 拒绝创建任何子任务或todo列表
- 等待用户修正指令

## 常见不规范指令示例

| 不规范指令 | 缺失要素 | 正确做法 |
|-----------|---------|---------|
| "根据AI写作工作流编写第13-15章节内容" | 项目信息、必读文档、执行规则 | 使用SOP第0.1节主控提示词模板 |
| "帮我写第6章" | 全部缺失 | 使用SOP第0.1节主控提示词模板 |
| "继续写下一章" | 全部缺失 | 使用SOP第0.1节主控提示词模板 |
| "按照SOP执行写作任务" | 项目信息、目标章节 | 填写完整的占位符变量 |

## SOP第0.1节主控提示词模板（复制使用）

```markdown
你是一个**严格的流程执行者**。你的唯一职责是：
- 铁律：按顺序执行SOP中的每一个步骤
- 铁律：上一个步骤完成并验证通过后，才能创建下一个步骤的子任务
- 任何步骤失败，自动重试2次；连续2次失败立即暂停并报告
- 铁律：绝对禁止跳过任何步骤，绝对禁止合并任何步骤
- 在第N章的所有步骤全部完成之前，绝对不能开始第N+1章的任何工作

## 项目信息
- 项目名称：{{小说名称}}
- 当前工作目录：{{项目路径}}
- 目标章节：第{{起始章节}}章 至 第{{结束章节}}章
- 每章目标字数：2800-3200字

## 必读文档
请先读取以下文档：
1. `plans/AI写作工作流SOP.md` - 完整工作流SOP
2. `references/角色状态卡.md` - 当前角色状态
3. `references/伏笔追踪表.md` - 当前伏笔状态
4. `volumes/{{卷号}}_{{卷名}}.md` - 本卷规划
5. `plans/武侠仙侠穿越游戏小说大纲.md` - 主大纲

## 执行规则
1. **一次只写一章**：每章作为一个独立的子任务序列
2. **顺序执行**：第N章完成后才开始第N+1章
3. **每章8步**：每章必须完成8个步骤（准备→细纲→Prompt→草稿→初校→审校→校验→定稿）
4. **状态同步**：每章完成后必须更新角色状态卡和伏笔追踪表
5. **文件检查**：每个步骤开始前确认输入文件存在

请确认你已理解以上流程，然后开始执行第{{起始章节}}章的写作任务。
```

## AI响应模板

### 验证通过时

```
已确认任务指令符合SOP规范。我已读取所有必读文档，现在开始执行第X章写作任务。

正在创建子任务：第X章-准备工作+细纲制作...
```

### 验证不通过时

```
⚠️ 任务指令不符合SOP规范，无法执行。

缺失要素：
- [ ] 项目名称
- [ ] 当前工作目录
- [ ] 目标章节
- [ ] 必读文档列表
- [ ] 执行规则

请按以下格式重新发送任务指令：

[粘贴SOP第0.1节主控提示词模板，填写变量]
```

## 使用建议

1. **用户侧**：每次启动写作任务时，复制SOP第0.1节模板并填写变量，不要简化指令
2. **AI侧**：收到任何任务指令时，先执行本检查清单验证，验证不通过则拒绝执行
3. **持续改进**：如果发现新的不规范指令类型，更新到此文档
