## 场景拆分

### 场景1：辞行

| 项目 | 内容 |
|------|------|
| 场景类型 | **C（悬疑）** |
| 时间地点 | 清晨，十里铺村 |
| 出场角色 | 李长风、赵老汉 |
| 核心动作 | 天刚亮。李长风走在去赵老汉家的路上。行囊已收拾好，干粮和水囊挂在腰间。清晨的雾气还没有完全散去，村道上湿漉漉的。路上遇到王婶，她正拿着竹扫帚扫院门前的土路。李长风跟她打招呼，她应了一声，继续扫。扫帚从右往左，三下，停一下，再从右往左，三下，停一下。节奏均匀。李长风没在意，继续往前走。到了赵老汉家，辞行。 |
| 对话要点 | - 赵老汉：（看到李长风来）"后生，要走了？" - 李长风："嗯。去黑风寨。" - 赵老汉：（沉默片刻）"路上小心。那地方不太平。" - 李长风："知道。" - 赵老汉：（欲言又止，沉默了一会儿）"今年的收成比往年差了三成。" - 李长风：（看着他） - 赵老汉：（摆摆手）"走吧。赶路要紧。" |
| 过渡方式 | 赵老汉的话让李长风起了疑。走出赵老汉家时，他想起刚才路上遇到的王婶，那个扫地的节奏。他停下来，站在路边看。王婶还在扫。从右往左，三下，停一下。再从右往左，三下，停一下。跟他刚才看到的一模一样。不是他停下来看的那一会儿。是节奏没变过。 |
| 场景细节要求 | 1. 开头直接写他在路上，不要写收拾行装 2. 王婶的行为要写得日常：扫地、均匀的节奏，不是"扫三下走五步"那种刻意 3. 赵老汉的话要隐晦：用"村里的路也不好走"代替"别多想" 4. 赵老汉说完后摆摆手，不再解释 5. 李长风的反应：不是震惊，是回头看、观察、确认 6. ⚠️ 避免禁用语："心中起了疑""眼中闪过异色" 7. ⚠️ 林婉儿不出现 8. 场景1聚焦"辞行+初疑"，不展开测试 |

### 场景2：测试

| 项目 | 内容 |
|------|------|
| 场景类型 | **C（悬疑加深）** |
| 时间地点 | 上午到中午，十里铺村各处 |
| 出场角色 | 李长风、村民若干（赵铁匠、刘寡妇、王老猎户等） |
| 核心动作 | 李长风没有立刻离开。他在村里走了一圈。村东的铁匠铺，赵铁匠在打铁。锤起锤落。打三下，翻一次铁，再打三下，放进水里淬火。嗤——蒸汽冒起。拿出来，再打三下，翻一次铁。节奏不变。李长风站在铺子外面数了十轮，一次没差。村西的刘寡妇在院门口晒豆子。把豆子从竹匾里倒出来，摊平，用手翻一次，再摊平。每隔一段时间，做一次。李长风看了三遍，动作完全一样。他走到村口老槐树下。王老猎户坐在树根上抽旱烟。烟袋锅子磕三下，装烟，点火，吸三口，磕烟灰。重复。李长风走过去，站在他面前，问："王伯，抽早烟呢。"王老猎户抬起头，看了他一眼。"嗯。"然后烟袋锅子磕三下。装烟。点火。吸三口。磕烟灰。仿佛刚才的对话没有发生过。李长风站在原地，看着这三个人。他们的节奏不一样。但有一个共同点：动作在重复。路线固定。次数固定。间隔固定。 |
| 对话要点 | - 李长风：（对王老猎户）"王伯，抽早烟呢。" - 王老猎户：（抬头看他一眼）"嗯。" - 王老猎户：（磕三下，装烟，点火，吸三口，磕烟灰） - （无其他对话） |
| 过渡方式 | 李长风从村口走回来。他没有再跟任何人说话。他站在村中央，看着四周。阳光很好，鸡鸣狗吠。一切都很正常。太正常了。正常得不像是真的。他想起赵老汉说的话："村里的路，也不好走。" |
| 场景细节要求 | 1. 每个村民的行为要写得更日常：打铁淬火、晒豆子翻豆子、抽旱烟，都是正常农活 2. 重点是"精确重复"：十轮打铁一次没差，晒豆子动作完全一样 3. 王老猎户的对话：回应了，但立刻回到固定节奏，不是"仿佛对话没发生过"这种解释性文字 4. 测试的方法是"数轮次"，不是"同时观察"——李长风一个一个看，每次确认节奏不变 5. 环境描写：阳光、鸡鸣、狗吠，一切看似正常，配合悬疑感 6. ⚠️ 不要写村民是傀儡，只写行为异常 7. ⚠️ 赵老汉的话要呼应："村里的路也不好走" 8. ⚠️ 避免禁用比喻："如同一台机器""像被线牵着的木偶" 9. 场景2专注"观察+测试+确认"，与场景1区分 |

### 场景3：离开

| 项目 | 内容 |
|------|------|
| 场景类型 | **F（转折冲击）** |
| 时间地点 | 中午，十里铺村村口 |
| 出场角色 | 李长风 |
| 核心动作 | 李长风站在村口。行囊在肩上，剑在腰间。身后是十里铺村。面前是通往外面的土路。他最后看了一眼村里。铁匠铺的叮当声。扫帚扫地的沙沙声。一切还在继续。他转过身，踏上土路。走了十步。停下来，回头。铁匠铺的叮当声停了。扫帚声停了。村口的狗叫声也停了。整个村子安静了一瞬。然后——叮当。沙沙。汪。声音又响起来了。节奏跟之前一样。仿佛刚才的安静只是错觉。李长风站在土路上，看着远处的村子。他等了一炷香的时间。声音没有变。他又走了百步。再次回头。十里铺村的轮廓在远处，炊烟升起，跟其他村子没有两样。但他知道不一样。他转过身，加快脚步。行囊在肩上颠着，剑在腰间晃着。他不再回头。 |
| 对话要点 | （无对话，纯行动描写） |
| 过渡方式 | 李长风加快脚步，沿着土路向前走去。身后的十里铺村渐渐消失在视野中。系统面板在此时弹出。 |
| 场景细节要求 | 1. 村口的环境：土路，两旁的野草，远处的群山 2. 转折瞬间：所有声音同时停了一瞬，然后同时恢复——这是真正的"转折冲击" 3. 不要解释"为什么停"，只写"停了又响" 4. 李长风的心理活动要简洁：短句，不分析，不感慨 5. 声音的描写：叮当（打铁）、沙沙（扫地）、汪（狗叫） 6. 回头的动作：两次回头，第一次确认声音停止，第二次确认村子外观正常 7. 系统面板弹出内容：【任务三：离开十里铺村，前往黑风寨】【进度：已离开十里铺村】 8. ⚠️ 不揭露真相，只写异常 9. ⚠️ 结尾不要写"一切才刚刚开始"之类的话 10. ⚠️ 结尾用动作结束：他不再回头 11. 场景3聚焦"转折+离开"，是真正的F型转折 |

## 特殊要求

1. **悬疑感的写法**：不要直接说"这很诡异"，而是通过具体的行为描写让读者自己感到不对劲。
2. **村民行为的描写**：要写得很日常（打铁、晒豆子、抽旱烟），但精确重复的模式让读者产生不安感。
3. **赵老汉的暗示**：用"村里的路也不好走"这种双关语，不要直接提示异常。
4. **节奏控制**：场景1（辞行+初疑）→场景2（测试确认，悬疑加深）→场景3（声音同时停止，转折冲击）。从平静到悬疑到不安。
5. **禁用语检查**：特别注意避免"心中一惊""眼中闪过异色""倒吸一口凉气""如同傀儡""像被线牵着"等高频AI痕迹。
6. **不揭露真相**：本章只是埋设伏笔，不要解释为什么村民会这样，不要写系统或高维文明。
7. **结尾要克制**：不要写太多感慨，用"他不再回头""路还长"这样的短句和行动结束。
8. **场景1和场景2功能区分**：场景1聚焦辞行和初疑（王婶扫地节奏），场景2专注测试确认（数轮次观察多人）。
