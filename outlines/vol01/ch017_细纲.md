### 场景一：石阶深入（约1000-1200字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境氛围 + C悬疑推理 |
| 时间地点 | Day 6 上午，地下石阶，从遗迹石屋后壁裂缝往下 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人沿石阶向下深入，环境逐渐变化，发现墙壁上的古老壁画/符文 |
| 环境描写 | 石阶约三十余级，越往下走空气越潮湿，带着土腥味和某种说不清的陈旧气息。石阶两侧的灰白色菌丝越来越密集，从新鲜的水汽逐渐变为干燥的灰白色，像是年代越久远颜色越浅。到第十五六级时，菌丝消失了，石壁上开始出现人工凿刻的痕迹 |
| 壁画/符文 | 石阶两侧的石壁上出现壁画——不是绘画，而是用某种锐器凿刻出来的浅浮雕。内容模糊，但能辨认出：①一群人站在一座建筑前，建筑顶部有云纹图案；②中间一人手持长剑，剑尖指向地面，剑柄处刻着一个"云"字；③最后一组画面：建筑崩塌，人四散奔逃，地面裂开，有某种东西从裂缝中涌出 |
| 李长风辨认 | 李长风注意到壁画的风格与游戏《剑道天下》中"归元宗"门派的建筑风格高度一致——尤其是云纹图案，与游戏中归元宗山门的标志性装饰完全对应。他心中升起一种奇怪的感觉：游戏里的归元宗场景，可能就是以这个地方为原型设计的 |
| 林婉儿剑纹 | 从石阶第十级开始，林婉儿手背上的剑形纹路开始发热，温度持续升高。到壁画出现时，已经明显发烫。她将右手缩入袖中，指尖微微蜷曲，但没有出声。李长风注意到她的异常，回头看了一眼，她没有说话，只是摇了摇头 |
| 灵气异常 | 越往下走，空气中的某种"异样感"越强。李长风作为引气入体的修士，能感觉到空气中弥漫着一种与外界不同的灵气——不是瘴气那种灰绿色的、带有腐蚀性的气息，而是更纯净的、但极其微弱的灵气。这种灵气不是自然散逸的，而是被某种力量引导、聚集在这里的 |
| 过渡方式 | 石阶走到尽头，眼前出现一个宽阔的地下空间 |

### 场景二：地下大厅（约1200-1500字）

| 项目 | 内容 |
|------|------|
| 场景类型 | B动作主导 + C悬疑推理 |
| 时间地点 | Day 6 上午→中午，地下大厅（遗迹核心区域） |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 到达地下大厅，发现云崖子遗骸和相关遗物，获取关键信息 |
| 大厅环境 | 石阶尽头是一个约三丈见方的地下大厅，顶部有裂隙，少量天光从上方瘴气中透下来，形成几道灰绿色的光柱。大厅地面铺着完整的青石板，中央有一个已经干涸的圆形水渍痕迹——可能是某种阵法或水池。四周墙壁上有四个壁龛，其中三个空着，一个里面有物品 |
| 残缺阵法 | 大厅地面上有明显的阵法痕迹——以中央水渍为中心，向外延伸出九条浅沟，呈放射状排列，每条浅沟的末端有一个小凹槽。阵法的大部分已经被岁月腐蚀得模糊不清，但基本结构还在。李长风辨认出这是一个"锁灵阵"——与石屋中的"聚灵阵"配合使用，将灵气锁定在特定区域。阵法已经失效，但残留的阵纹还在微微发光 |
| 云崖子遗骸 | 在壁龛下方的地面上，靠墙坐着一具遗骸。遗骸穿着残破的道袍，道袍的领口和袖口有云纹刺绣——与铜牌上的云纹一致。遗骸保持坐姿，背靠墙壁，左手握着一柄剑（剑身已锈蚀大半），右手搭在膝上，指骨间夹着一枚玉简。遗骸的头微微低垂，像是在沉思，又像是在等待 |
| 遗物检查 | 李长风蹲下身，检查遗骸和周围。遗骸旁边有一个破损的储物袋（已风化，内容物散落），里面有几块碎裂的灵石（灵气耗尽，变成普通石头）、两枚失效的符箓、一本残破的笔记（纸张脆化，一碰就碎）。李长风小心地拿起玉简 |
| 玉简信息 | 玉简入手冰凉，表面有细微的裂纹。李长风用真气试探——玉简有反应，微微发亮。他将玉简贴在额头，一股信息流入脑海："瘴非天降，乃人所为。归元宗内门弟子云崖子，奉命封印瘴眼，同伴尽殁，唯余一人。瘴眼之下，有物未明，不可轻动。若后来者见此简，切勿——"信息到此中断，像是被什么力量强行切断 |
| 机关触发 | 李长风取下玉简的瞬间，大厅地面微微震动。中央干涸的水渍痕迹中，九条浅沟同时亮起微弱的灰绿色光芒——不是灵气，而是瘴气。阵法被触发了，但不是原来的锁灵阵，而是某种被瘴气污染的变异阵法。灰绿色的瘴气从浅沟中涌出，向四周扩散 |
| 小冲突 | 瘴气从地面涌出，形成低矮的雾气层，约到脚踝高度。雾气中开始出现某种东西的轮廓——不是异兽，而是阵法残留的"影像"：几个模糊的人形在瘴气中行走，动作僵硬，像是被什么力量操控。李长风拉着林婉儿后退，避开瘴气覆盖区域。影像没有攻击性，只是在原地重复某种动作——像是当年的封印场景被阵法记录下来 |
| 过渡方式 | 李长风将铜牌、玉简和能带走的物品收入行囊，准备撤离。但大厅顶部开始出现裂缝，碎石落下——遗迹开始不稳定 |

### 场景三：危机与收获（约800-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | F转折冲击 + B动作主导 |
| 时间地点 | Day 6 中午，地下大厅→石阶→地面 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 遗迹开始崩塌，两人携带关键物品撤离，回到地面后发现瘴气有变化 |
| 崩塌触发 | 李长风将玉简收入贴身衣袋的瞬间，大厅顶部的裂缝扩大。不是自然崩塌——是阵法被触发后的连锁反应。瘴气污染了原本的稳定结构，遗迹的能量平衡被打破。碎石从顶部落下，砸在地面上发出沉闷的响声 |
| 撤离过程 | 李长风拉着林婉儿往石阶方向跑。右腿每落地一次都传来剧烈刺痛，但他没有停。林婉儿跟在身后，手背剑纹烫得厉害，但她咬紧牙关没有出声。石阶上的壁画开始脱落，凿刻的痕迹在崩塌中碎裂。跑到第十五六级时，一侧石壁塌陷，碎石滚落，险些堵住通路 |
| 关键物品 | 撤离时李长风携带的物品：①铜牌（已有一件）；②玉简（新获得，含云崖子遗言）；③遗骸旁储物袋中的残破笔记（只来得及抓了几页，纸张脆化但字迹还能辨认） |
| 回到地面 | 两人从石屋后壁裂缝爬出，回到地面。石屋在身后发出沉闷的响声，裂缝扩大，石阶入口被塌陷的石块封住。遗迹彻底封闭 |
| 瘴气变化 | 回到地面后，李长风注意到周围的瘴气有变化——比进入石阶之前淡了一些，颜色从灰绿色变为浅灰色，浓度降低了约三分之一。像是遗迹封闭后，瘴气的"源头"被削弱了 |
| 新悬念 | 林婉儿低头看右手。手背上的剑纹温度在撤离过程中达到最高点，此刻正在慢慢消退。但剑纹的颜色——原本只是皮肤下的淡白色印记——现在变成了极浅的银灰色，像是吸收了遗迹中的某种能量。她将手缩入袖中，没有让李长风看到 |
| 过渡方式/结尾 | 李长风站在石屋前，望着被封住的裂缝，手中握着那几页残破的笔记。他低头翻看，其中一页上有熟悉的字迹——与铜牌上的楷书一致，写着："归元宗内门，云崖子，瘴眼之下——"后面的字被污渍覆盖，无法辨认。远处，遗迹外围方向传来异兽的低吼声，比之前更远了 |

## 结尾钩子

- **钩子类型**：信息悬念+状态异常
- **具体内容**：两人获取了云崖子的玉简和笔记残页，但信息不完整（"切勿——"后中断）。遗迹封闭，瘴气变淡，暗示遗迹与瘴气源头有关。林婉儿手背剑纹颜色变化（白→银灰），她没有告诉李长风
- **预期效果**：读者对"瘴眼之下"的内容产生好奇，对剑纹变化感到疑惑，想要知道后续发展
