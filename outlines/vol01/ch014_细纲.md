## 场景拆分

### 场景一：暮色赶路（约900-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑探索 |
| 时间地点 | 傍晚至入夜，青州镇以北官道→小路 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人离开茶摊后沿官道往北走，根据陈文远提供的方向在岔路口转入小路。李长风伤势影响行进速度，林婉儿刻意放慢脚步配合。途中两人简短交谈，讨论断魂谷是否值得去 |
| 对话要点 | ①李长风："陈文远说谷里有怪声，像有人在哭。你觉得是什么？"——他问林婉儿的判断；②林婉儿不直接回答，反问："你信他说的？"——她一贯用问句引导；③李长风分析："三拨人进去没出来，如果是普通的山贼或野兽，不会一个都跑不出来。"——理性推理；④林婉儿："所以你更要去看。"语气不是关心，是确认；⑤李长风："去看看。"——确认决定 |
| 过渡方式 | 天色完全暗下来，两人点起火把（林婉儿从医箱里取出火折子），转入一条杂草丛生的小路。环境变化自然过渡到下一场景 |
| 场景细节要求 | ①李长风右腿落地时微跛，每走百步就要调整一次重心，但他不表现出来；②林婉儿走路步子均匀，医箱斜挎在肩上，火把的光映在她脸上，神色平静；③官道越来越窄，两旁的树木越来越密，远处的天空被一层灰蒙蒙的雾气笼罩——那就是断魂谷方向的瘴气；④路上遇到一个废弃的路标，木牌上刻着"断魂谷"三字，字迹模糊，像是很多年前刻的；⑤偶尔有夜鸟叫，声音尖锐，但不像"有人在哭" |

### 场景二：谷口初探（约1000-1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境氛围 |
| 时间地点 | 入夜，断魂谷谷口 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人抵达断魂谷谷口，实地观察环境。瘴气、怪声、地形异常逐一印证陈文远的描述。林婉儿展示超出常人的观察力 |
| 对话要点 | ①李长风描述所见：谷口两侧是峭壁，中间一条窄路，瘴气从谷内弥漫出来，带着腐臭味；②林婉儿蹲下检查地面："有人走过的痕迹，不止一拨。"——她发现地面有不同脚印；③李长风："进去多久了？"——他问时间；④林婉儿用树枝拨开落叶："最早的三四天，最近的……不到一天。"——她通过脚印的新鲜程度判断时间；⑤此时传来一阵声音——"呜……呜……"，像有人在哭，但断断续续，不是真的人声 |
| 过渡方式 | 怪声传来时，李长风手背剑纹微微发热。他低头看手，发现剑纹比平时亮了一些。这个异常现象推动两人继续深入探索，过渡到场景三 |
| 场景细节要求 | ①瘴气描写：灰绿色，像一层薄雾贴在谷口地面，高度约到膝盖，不往上飘；②腐臭味：不是尸臭，更像是"什么东西在腐烂但又没完全烂透"的味道；③怪声：低沉、断续、有规律（约每半分钟响一次，每次持续3-5秒），不像人的哭声，更像是"风吹过某个空洞"的声音；④林婉儿检查地面时手指很稳，没有犹豫，说明她有野外追踪经验；⑤李长风手背剑纹发热时，他把袖子拉下来遮住，不想让林婉儿看到；⑥谷口有一块大石头，上面有刀劈的痕迹——像是有人在这里打过架 |

### 场景三：前人遗物（约800-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑发现 |
| 时间地点 | 断魂谷谷口内侧十余步 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人沿谷口窄路往里走了十余步，发现前人遗留的物品——一把断刀、半截猎弓、一只草鞋。这些物品的主人不在附近，暗示进入谷内的人遭遇了不测 |
| 对话要点 | ①李长风捡起断刀："刀口是崩断的，不是砍断的。"——分析断裂方式；②林婉儿："不是被人砍断的，是被什么东西咬断的。"——她发现刀口有齿痕；③李长风检查猎弓："弓弦是自己断的，不是拉的。"——弓没有使用痕迹；④林婉儿把草鞋翻过来看："鞋底有血，但鞋上没有脚印。"——说明这人光着脚走了，或者……被拖走了；⑤李长风沉默。他把手里的东西放下，看了一眼前方被瘴气笼罩的谷内深处；⑥林婉儿："还要进去？"——她问，语气依旧平静；⑦李长风："明天。"——他决定不连夜进去，需要做准备 |
| 过渡方式 | 两人退回谷口外，找了一块地势较高的地方扎营。李长风坐在火堆旁，看着手里的断刀，脑中反复回想齿痕的形状。结尾钩子：火光照耀下，李长风发现断刀的刀柄上刻着一个字——"云"。与他矿洞获得的铜柄断剑上的古篆字相同 |
| 场景细节要求 | ①断刀：约两尺长，刀身锈迹斑斑，刀口有不规则的锯齿状缺口，像是被某种利齿咬断的；②半截猎弓：只剩弓臂的一半，弓弦松垮地垂着，弓身上有爪痕——不是刀砍的，是某种动物的爪子抓的；③草鞋：草编的，鞋底有暗红色干涸血迹，鞋面上没有血，说明血是从脚底来的；④三件物品散落在方圆三步范围内，像是主人仓皇逃跑时掉落；⑤李长风发现"云"字时，手背剑纹又热了一下，但这次林婉儿没注意到；⑥结尾时瘴气已经漫到了他们扎营的地方，脚下的草开始发黄 |

## 结尾钩子

- **钩子类型**：重大发现+悬念
- **具体内容**：李长风发现断刀刀柄上刻着"云"字，与他在矿洞获得的铜柄断剑上的古篆字相同。这意味着断魂谷内失踪的人与矿洞枯骨可能有某种关联
- **预期效果**：读者好奇"云"字的含义，断魂谷内的真相是什么，失踪的人遭遇了什么，期待下一章的深入探索

## 特殊要求

1. **环境描写差异化**：瘴气不要写成"白色的雾气"或"朦胧的烟雾"，要写出"灰绿色、贴地、带腐臭味"的具体特征。避免"月光如水""空气中弥漫着"等AI高频环境描写
2. **林婉儿能力展示**：她判断脚印时间和血迹来源时要自然，像在说一件平常事，不要写成"她眼中闪过一丝精光"或"她心中暗道"。用行为展示能力，不要心理描写
3. **"云"字回收**：ch004矿洞断剑上的"云"字在本章场景三结尾出现第二次，形成关联。这个发现要让读者产生"原来如此"的感觉，但不要解释"云"字到底意味着什么
4. **节奏控制**：本章是探索的准备阶段，不直接进入战斗。以发现"云"字断刀作为结尾钩子，给读者足够的悬念，同时为ch015-16留出空间
5. **禁用语检查**：严格检查禁用语列表，特别是"然而""与此同时""嘴角勾起""眼中闪过""他心中暗道""倒吸一口凉气"等
6. **感情温度计**：本章感情温度维持在18，不主动推进。两人之间的互动是合作探索式的，不涉及私人情感
7. **场景类型轮换**：本章C→E→C，与ch013（D→B→A）不冲突，与卷规划建议的C→E一致（场景三增加C悬疑发现）
