## 场景拆分

### 场景1：断剑

| 项目 | 内容 |
|------|------|
| 场景类型 | **B（战斗 aftermath）** |
| 时间地点 | 夜里，李长风的茅屋 |
| 出场角色 | 李长风 |
| 核心动作 | 夜深了，村里安静下来。李长风独自坐在屋里，把缺损的铁剑放在桌上。剑刃的缺口在油灯下清晰可见。他试着挥了几下，剑身发沉，重心已经偏了。这把剑撑不过下一场战斗。他把铁剑放回桌上，又摸出腰间的铜柄断剑。柄上的古篆"云"字在灯光下泛着暗金色的光。两把剑，都断了。他需要一把新剑。不是现在这种凑合用的，而是一把能陪他走下去的剑。就在他伸手去拿铜柄断剑的一瞬间，指尖触到剑柄上"云"字的凹槽。指尖传来一阵微弱的震颤。不是剑在动。是手背上的剑纹在动。他低头看去，手背上的剑形纹路正在发烫。纹路从手背延伸到掌心，像是一条线，连向丹田。真气，从丹田涌出，沿着那条线，流向掌心。掌心碰到铜柄断剑的瞬间，"云"字亮了一瞬。然后暗了下去。 |
| 对话要点 | （内心独白为主）  - "剑废了。"  - "两把都断了。"  - "需要一把新的。"  - （触到"云"字时）"什么？"  - （手背发烫时）他没有抽回手。 |
| 过渡方式 | 手背剑纹的异象持续了几息后消退。李长风收回手，剑纹恢复了原本的样子。但那股真气流动的感觉，还在。 |
| 场景细节要求 | 1. 夜里的安静与白天的混乱形成对比 2. 油灯的光要具体：昏黄、摇曳、照在剑刃缺口上 3. 试剑的动作要具体：挥剑、感受重心、判断是否能用 4. 铜柄断剑的描写：古篆"云"字、暗金色的光泽、系统无法识别 5. 手背剑纹发烫要写得具体：不是"一股暖流"，是真实的生理感受 6. 真气流动的路线要具体：丹田→手背剑纹→掌心 7. "云"字亮了一瞬要写得克制，不要写得太夸张 8. ⚠️ 避免禁用语："一股暖流""心中一惊""眼中闪过精光" |

### 场景2：剑鸣

| 项目 | 内容 |
|------|------|
| 场景类型 | **E（环境氛围）** |
| 时间地点 | 夜里到凌晨，十里铺村 |
| 出场角色 | 李长风 |
| 核心动作 | 那股真气流动的感觉没有消失。它留在了掌心，像一个印记。李长风闭上眼，感受那股真气。它不再像之前那样散乱。它在掌心里汇聚，凝成了一个点。那个点，和手背上的剑纹对上了。他睁开眼。桌上的两把断剑，铜柄断剑和无名铁剑，都在。但它们的"气"不一样。铜柄断剑的气是沉寂的，像一块死铁。无名铁剑的气是散的，像一堆碎片。而他掌心的那个点，是活的。他伸出手，不是去拿剑，而是去感受。掌心对着两把剑，真气从那个点涌出，像水波一样扩散。铜柄断剑没有反应。无名铁剑的剑刃，轻微地颤了一下。不是风吹的。是剑在回应他的真气。他收回手。剑刃恢复了平静。但他知道，那不是错觉。他又伸出手。这一次，他试着把真气"推"向剑刃。掌心的真气点收缩，然后释放。真气触到剑刃的瞬间，剑刃发出一声低鸣。不是金属碰撞的声音。是剑自己在响。像是一种呼唤。 |
| 对话要点 | （无对话，纯行动描写）  - 感受真气的凝集  - 试探两把剑的反应  - 第二次试探，真气推向剑刃  - 剑鸣响起 |
| 过渡方式 | 剑鸣声在夜里传出去，很快消散。李长风收回手，掌心的真气点还在。他知道了两件事：第一，他的真气可以和剑产生共鸣。第二，那把无名铁剑，虽然缺口累累，但它在回应他。 |
| 场景细节要求 | 1. 环境描写要配合氛围：夜里的寂静、油灯的微弱光线、远处的狗吠 2. 真气凝聚的过程要具体：散乱→汇聚→凝成点 3. 两把剑的"气"的对比：铜柄断剑沉寂 vs 无名铁剑散乱 4. 剑鸣声要写得独特：不是"一道寒光"，是低沉的、金属内部的共鸣 5. ⚠️ 不要写得太玄幻，要保留武侠的真实感 6. ⚠️ 避免禁用比喻："如同一头沉睡的猛兽""宛如利剑出鞘" 7. 李长风的反应要冷静：不是震惊，是分析、确认 |

## 特殊要求

1. **剑道天赋初显的写法**：不要写主角突然会了什么剑法，而是写他能感受到剑的"气"，能与剑产生共鸣。这是天赋，不是技能。
2. **真气觉醒的写法**：不要写"一股暖流"，要写具体的生理感受：丹田发热、真气沿特定路线流动、掌心与剑产生共鸣。
3. **无名铁剑的"认主"**：不是玄幻的认主仪式，是主角的真气与剑产生了共鸣，剑变得"合手"。外表不变，但内在变了。
4. **节奏控制**：场景1（战后独处，发现异象）→场景2（深夜感受，剑鸣回应）→场景3（清晨决意，系统确认）。从安静到热血到期待。
5. **禁用语检查**：特别注意避免"一股暖流""心中一惊""眼中闪过精光""嘴角勾起""倒吸一口凉气"等高频AI痕迹。
