| 时间地点 | 同场景一，拖痕前方十余步处 |
| 出场角色 | 李长风、林婉儿、异兽（首次出现） |
| 核心动作 | 咕噜声越来越近，瘴气突然向两边分开，一只异兽冲出来。体型似犬但大了一圈，约三尺高，皮毛灰褐色，与瘴气颜色接近。四爪着地，爪尖锋利，行动时几乎无声。异兽没有立即攻击，而是停在五步外盯着两人。李长风握剑戒备，异兽突然暴起扑向李长风。李长风侧身让过，无名铁剑斜劈，剑刃砍中异兽侧腹，入手感觉不对——不像砍中血肉，更像是砍进了什么致密坚韧的东西。异兽吃痛后退，侧腹有一道剑伤，但伤口不流血，反而渗出一丝灰绿色的雾气。异兽低吼一声，再次扑来，这次目标是林婉儿 |
| 对话要点 | ①李长风低喝："退后！"——对林婉儿，不是慌乱，是提醒；②林婉儿没退，反而从医箱里摸出一把短刃，语气平静："它怕火。"——昨夜扎营时异兽没靠近火堆，她注意到了；③李长风："知道。"——他昨晚也想过这事，但此刻白天无火可用；④异兽扑向林婉儿时，李长风跨步上前，铁剑横扫，剑刃再次砍中异兽前肢，这次看清了——异兽的皮毛下面不是血肉，而是一层致密的、灰绿色的、像菌丝一样的东西，剑刃砍进去时被卡住一瞬 |
| 过渡方式 | 异兽被两剑砍伤，低吼一声，突然转身窜进瘴气里，消失不见。但战斗未结束，异兽不是逃跑，而是绕到侧翼再次扑来 |
| 场景细节要求 | ①异兽外形：似犬但更大，三尺高，灰褐色皮毛与瘴气颜色接近，四爪锋利如钩，眼睛浑浊发黄，没有瞳孔感；②异兽行动：几乎无声，只在暴起时爪子抓地发出"咔"的一声，速度极快但转向不灵活；③战斗过程：李长风右腿旧伤被牵扯，落地时微跛，影响闪避速度，第一次侧身让过但慢了半拍，左臂被异兽爪子擦过，添了一道新伤（不深，但流血）；④剑伤异常：伤口不流血，渗出灰绿色雾气，像异兽体内不是血液而是瘴气；⑤林婉儿没闲着，她观察异兽行动轨迹，指出"转向不灵"，李长风利用这一点调整站位；⑥战斗不超过8个回合，干脆利落，李长风以伤换命，第三剑砍中异兽后颈，异兽惨嚎一声窜走；⑦战斗后李长风左臂新伤，右腿更痛，但无大碍 |

### 场景三：异象之谜（约800-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑发现 |

| 对话要点 | ①李长风："不是野兽。"——他看着灰绿色液体，"野兽的血不是这个颜色。"；②林婉儿："它的伤口在化雾。"——她回忆战斗中剑伤渗出雾气的情景，"那不是血，是瘴气。"；③李长风走到石壁前，手指抚摸刻痕："这上面有字。"——刻痕很深，但边缘被腐蚀了，只能辨认出几个字的轮廓；④林婉儿凑过来看，辨认出两个字："断……魂……"——与谷口木牌上的字一样；⑤李长风沉默。他把手缩回来，看了看前方更浓的瘴气；⑥此时远处传来几声低吼，不是一只，是多只异兽在呼应 |
| 过渡方式 | 多只异兽的低吼声从瘴气深处传来，越来越近。李长风看了一眼林婉儿，两人对视一瞬。结尾钩子：李长风握紧铁剑，说了一句"走"，不是退，而是往更深处走——因为退路方向也传来了异兽的叫声 |
| 场景细节要求 | ①灰绿色液体：不是血，粘稠度比水高，滴在草叶上草叶立刻发黄枯萎；②毛发溶解：灰褐色毛发碰到液体后表面起泡，像被酸腐蚀，几息后化成一缕灰烟散开，无残留；③石壁开凿痕迹：表面平整，有明显的凿击点，间距均匀，不是天然形成的；④刻痕字迹：被瘴气腐蚀严重，只能辨认出"断魂"二字轮廓，其余模糊不清，刻痕风格与谷口木牌不同，更像是用刀剑一点点刻出来的；⑤异兽低吼声：不是一只，至少三只以上，从不同方向传来，形成包围态势；⑥结尾时李长风左臂新伤用布条简单包扎，右腿更痛但咬牙忍住；⑦林婉儿手背剑纹在袖中微微发热，与昨夜听到怪声时类似但更弱 |

## 特殊要求

1. **异兽描写差异化**：不要写成"妖兽""魔兽"或"妖怪"，用"异兽"或直接描述外形。异兽不是修仙世界的标准妖兽，而是与瘴气共生的变异生物，体内是瘴气不是血液，尸体化雾消散。避免"狰狞""恐怖""令人胆寒"等AI高频词
2. **战斗代价体现**：李长风右腿旧伤和新伤必须影响战斗表现——闪避慢半拍、落地微跛、以伤换剑。不能写成"轻松碾压"或"毫发无伤"
3. **林婉儿能力边界**：她只负责观察和辅助，不出手战斗。指出"异兽怕火""转向不灵"即可，不要写成"她出手相助"或"她一剑刺中"。保持她隐藏实力的设定
4. **瘴气异常描写**：异兽伤口渗瘴气、尸体化瘴气、毛发溶解——这些异常现象要写得具体、有视觉冲击力，但不解释原因。让读者产生"这到底是什么"的疑问
5. **石壁刻字悬念**：刻字只辨认出"断魂"二字，其余模糊。不解释谁刻的、什么时候刻的、为什么刻。留下F019伏笔
6. **结尾钩子处理**：多只异兽包围，退路被断，李长风选择"走"而非"退"。这个决定要果断，不犹豫，体现角色性格
7. **禁用语检查**：严格检查禁用语列表，特别是"然而""与此同时""嘴角勾起""眼中闪过""他心中暗道""倒吸一口凉气""恐怖如斯"等
8. **场景类型轮换**：本章C→B→C，与ch014（C→E→C）不冲突（中间场景从E换成B增加动作戏），与ch013（D→B→A）也不同
9. **感情温度计**：本章感情温度从18提升到21，幅度小但真实。战斗中互相配合体现信任，但不涉及私人情感
10. **F018部分回收**：本章首次揭示异兽真相（齿痕、爪痕来源），但完整真相（异兽从哪来、为什么在谷内）留到后续章节
