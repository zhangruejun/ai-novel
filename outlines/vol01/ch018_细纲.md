### 场景一：残页解读（约900-1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑推理 |
| 时间地点 | Day 6 中午，瘴谷外围一处相对安全的岩洞，瘴气较谷内淡 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 李长风处理伤势的同时，仔细检查从遗迹带出的玉简和笔记残页 |
| 环境描写 | 岩洞不大，约一丈见方，洞壁有被水冲刷过的痕迹。洞口外的瘴气呈浅灰色，比进入遗迹前明显变淡，但仍贴着地面缓慢流动。洞内干燥，地上有前人留下的灰烬痕迹——不是最近留下的 |
| 伤口处理 | 李长风靠在洞壁上，卷起右腿裤管。膝盖处的伤口比之前深，布条与皮肉黏连，撕下时渗出新鲜的血。他没有皱眉，只是用牙齿咬住布条一端，一手按住伤口上方，一手猛地撕下。血珠渗出，他从林婉儿的医箱中取干净布条重新包扎。左臂的伤较浅，简单清理后换了布条 |
| 玉简检查 | 李长风取出玉简。玉简表面有细微裂纹，但整体完整。他再次用真气渡入，尝试获取更多信息——与之前一样，信息在"切勿——"处中断。他注意到一个细节：每次真气渡入，裂纹都会微微发亮，但光芒是灰绿色的，与瘴气颜色一致 |
| 笔记残页 | 李长风将数页笔记残页摊开在地上。纸张脆化发黄，但字迹大多可辨。其中三页内容较完整：①第一页记录了"瘴眼"的位置和封印方法——以锁灵阵配合九根镇灵钉，将瘴气源头锁定在地下；②第二页提到"同伴三人，先后被瘴气侵蚀，神智不清，自相残杀。唯我因修炼归元心法，暂保清醒"；③第三页字迹潦草，像是匆忙写就："瘴气非自然之物，乃是……"后面的字被污渍覆盖 |
| 李长风分析 | 他将三页笔记按时间顺序排列，结合玉简信息推断：归元宗发现了瘴气的真相→派人来封印→封印过程中出事→云崖子独自完成封印并留下警告。但"切勿——"和"瘴气非自然之物，乃是……"两处关键信息缺失 |
| 过渡方式 | 林婉儿在一旁沉默地观看，偶尔递上干净的布条或水。李长风注意到她的异常安静，开口询问 |

### 场景二：对话与试探（约900-1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | D对话/角色 |
| 时间地点 | Day 6 中午→下午，岩洞内 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人对话交流，李长风试探林婉儿对遗迹的反应，林婉儿隐瞒手背剑纹变化 |
| 对话要点 | 李长风问林婉儿在遗迹中是否有不适——她回答"只是觉得闷"。李长风没有追问，但目光在她右手袖口处停留了一息。林婉儿将右手往袖中缩了缩，转移话题："你觉得'瘴眼'是什么？"李长风回答："不是自然形成的东西。瘴气有规律地流动，像是被什么力量引导。" |
| 林婉儿的试探 | 林婉儿看似随意地问："你之前说，你在……梦里见过类似的地方？"（她用了"梦"这个模糊的词，没有直接说"游戏"）李长风沉默片刻，回答："不是梦。是一个……和我来处有关的地方。但细节不完全一样。"他没有多解释，林婉儿也没有追问 |
| 信息分享 | 林婉儿从医箱中取出两枚解毒丹，递给李长风一枚。她说："从医理上看，瘴气更像是某种……人为释放的毒素。自然形成的瘴气不会有这么强的腐蚀性，也不会持续这么久。"李长风接过解毒丹，没有立刻服用，而是收入行囊 |
| 剑纹异常 | 对话中，林婉儿右手不自觉地摩挲左手手背——这是她在压抑剑纹发热时的习惯性动作。李长风注意到了这个细节，但没有戳破。他只是说："休息一下，下午继续赶路。瘴谷不能久留。" |
| 过渡方式 | 两人短暂休整后准备离开岩洞。李长风将笔记残页小心收好，玉简贴身存放。刚走出岩洞，异变突生 |

### 场景三：瘴气异变（约1000-1200字）

| 项目 | 内容 |
|------|------|
| 场景类型 | A动作/冲突 |
| 时间地点 | Day 6 下午，岩洞外→瘴谷外围 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人离开岩洞后遭遇瘴气异变，被迫应对新的危机 |
| 异变触发 | 两人刚走出岩洞不到十步，地面传来轻微震动。不是遗迹崩塌那种剧烈震动，而是从地底深处传来的、持续的微颤。李长风停下脚步，蹲下身，手掌贴地——震源在下方极深处，方向与遗迹位置一致 |
| 瘴气变化 | 震动的同时，周围瘴气开始异变。原本已经变淡的浅灰色瘴气突然变得浓稠，颜色从浅灰转为暗灰，甚至有隐约的绿色在其中流动。瘴气流动速度加快，不再是贴着地面缓慢爬行，而是像被什么力量搅动，形成数个小型漩涡 |
| 李长风判断 | 李长风立刻拉住林婉儿往回退。他说："遗迹封闭后，瘴气源头失去了封印的压制。"不是遗迹削弱了瘴气——是遗迹封闭后，封印彻底失效了。之前的"变淡"只是短暂的能量真空期 |
| 应对策略 | 瘴气浓度在快速上升，超过了进入遗迹前的水平。李长风从行囊中取出那枚解毒丹，递给林婉儿一枚，自己服下一枚。解毒丹入腹后，胸口升起一股温热，但很快被瘴气的侵蚀感抵消。他说："不能停，往高处走。" |
| 撤离过程 | 两人向瘴谷外围的高地撤离。右腿伤口在快速移动中再次被扯开，血渗出新的布条，李长风步子微乱但没有停。林婉儿跟在他身后，右手始终缩在袖中——手背上的银灰色剑纹在瘴气异变后再次开始发热，温度比在遗迹中更高，但没有烫到无法忍受 |
| 异兽出现 | 撤离到一处坡地时，瘴气中出现异兽的轮廓。不是之前在瘴谷中遇到的那种——这只体型更大，约莫牛犊大小，外形像放大的蜥蜴，但背部有一排骨刺。它的眼睛是浑浊的黄色，盯着两人看了三息，没有立刻攻击 |
| 冲突处理 | 李长风拔出无名铁剑，横在身前。他没有主动进攻——右腿伤势不允许他进行大幅度移动。异兽也没有进攻，它只是盯着两人，然后缓缓后退，消失在瘴气中。李长风皱眉——异兽的退让不是畏惧，更像是……在观察 |
| 结尾钩子 | 两人继续撤离到高地。回头望去，瘴谷方向的瘴气已经浓到看不见地面，灰绿色的雾气形成一根粗壮的柱子，从地底直冲天空。柱子顶端有某种东西在闪烁——不是光，而是某种规则的、有节奏的脉动，像是心跳 |
| 过渡方式/结尾 | 李长风盯着那根瘴气柱，右手无意识地握住了胸前的玉简。玉简表面的裂纹中，灰绿色的光芒与瘴气柱的脉动频率一致。他低声说："瘴眼……开了。"林婉儿站在他身后，手背上的银灰色剑纹烫得几乎要烧穿皮肤。她将右手紧紧攥成拳，指甲陷入掌心 |

## 结尾钩子

- **钩子类型**：危机+新发现
- **具体内容**：遗迹封闭后封印失效，瘴眼重新开启，瘴气浓度超过之前。瘴气柱顶端出现有规律的脉动，玉简裂纹光芒与脉动频率一致，暗示玉简与瘴眼存在某种联系。林婉儿手背剑纹再次异常发热，她继续隐瞒
- **预期效果**：读者对瘴眼打开后的后果感到紧张，对玉简与瘴眼的联系产生好奇，想要知道两人接下来如何应对

## 特殊要求

- 伤口处理描写要简洁写实，不要过度渲染血腥
- 李长风撕布条的动作要干脆，体现他处理伤痛的经验
- 林婉儿的隐瞒要 subtle——不是刻意撒谎，而是选择不主动透露
- 瘴气异变要有层次感：震动→颜色变化→浓度上升→形成气柱
- 异兽的"观察"行为要写得诡异，让读者感到不安
- 结尾瘴气柱的脉动要有节奏感，暗示背后有某种规律或力量
- 玉简裂纹与脉动同步的细节不要直接点明，只在叙事层面呈现
