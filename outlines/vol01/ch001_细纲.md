## 场景拆分

### 场景一：苏醒（约900-1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境氛围 |
| 时间地点 | 清晨，茅草屋内，新手村边缘 |
| 出场角色 | 李长风（主角） |
| 核心动作 | 主角在剧烈头痛中苏醒，发现自己身处陌生的茅草屋内，身体虚弱且带有伤痕；原主记忆如潮水般涌入，两段记忆开始融合，产生认知混乱 |
| 对话要点 | 无对话，以内心独白和环境描写为主 |
| 过渡方式 | 门外传来脚步声和敲门声，打断主角的思绪 |

**场景细节要求**：
- 环境描写要突出"陌生感"：简陋的茅草屋、土墙、木床、窗外的鸟鸣和远处村落的声音
- 身体感受要真实：头痛欲裂、胸口发闷、左肩有钝痛感（原主被打伤的痕迹）
- 记忆融合过程要有层次：先是碎片化画面（村落的日常、一个模糊的女性身影），然后逐渐清晰（原主的姓名、身份、生活轨迹）
- 原主记忆中包含关键暗示：这个世界"不只是个游戏"（**F002伏笔埋设**，通过原主某次听老人讲述的传说暗示）
- **重要约束**：系统面板不出现（第3章才解锁）

### 场景二：送药（约900-1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | A对话推进 |
| 时间地点 | 清晨，茅草屋内 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 林婉儿推门而入，为主角送药；两人在对话中建立初步关系；主角注意到自己手背上浮现的神秘剑纹 |
| 对话要点 | 1. 林婉儿关切询问主角伤势（"你醒了？感觉怎么样？"）<br>2. 主角试探性回应，确认自己穿越的事实（通过林婉儿的外貌、服饰、用语判断）<br>3. 林婉儿提到主角受伤的原因（"那天晚上你从后山回来，浑身是伤……"）——**不详细说明，为主角留下悬念**<br>4. 林婉儿离开前叮嘱好好休息，暗示会再来 |
| 过渡方式 | 林婉儿离去，门关上后，门外传来隐约的说话声 |

**场景细节要求**：
- 林婉儿首次亮相的外貌描写：素色衣裙、清秀面容、眼神灵动但带着一丝不易察觉的深邃（**暗示身份不简单，为F005-1伏笔做铺垫**）
- 林婉儿的言行要"过于镇定"：见到主角苏醒没有表现出过度的惊讶，仿佛早有预料（**F005-1伏笔：穿越者见到穿越者异常镇定**）
- 手背剑纹的描写：青黑色纹路，形似一柄古剑的轮廓，隐隐发烫；主角不确定是原主就有的还是穿越后出现的
- 对话中要体现主角的试探心理：利用游戏经验判断林婉儿是否是NPC，但发现她的反应过于真实
- **重要约束**：林婉儿出场后，第2-11章将退场不出现，本章是她首次也是近期最后一次亮相

### 场景三：异变（约800-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑 |
| 时间地点 | 清晨，茅草屋内 |
| 出场角色 | 李长风（主角）、门外打手（声音出场） |
| 核心动作 | 林婉儿离去后，门外传来粗犷的说话声（"那小子死了没？"）；主角屏息倾听，随后脑海中突然涌入大量陌生记忆，不属于自己的人生经历在眼前闪现 |
| 对话要点 | 1. 门外打手声音（2-3人）："那小子死了没？""应该差不多了，那天下手那么重。""走吧，黑风寨的人还等着回话呢。"（**F010伏笔埋设：原主被打伤事件，打手为黑风寨探子**）<br>2. 主角内心反应：黑风寨？这是游戏中的新手村敌对势力！ |
| 过渡方式 | 记忆洪流涌来，本章在悬念高潮中结束 |

**场景细节要求**：
- 门外声音要"隐约但可辨认"：主角刚受伤，听觉可能不太清晰，但关键信息必须传达
- "黑风寨"三个字要给主角（和读者）强烈冲击：这是《剑道天下》游戏中新手村附近的山贼势力
- 记忆洪流的描写要有层次感：先是快速闪过的画面（原主的童年、少年、一次夜行后山、被袭击的瞬间），然后是情感冲击（恐惧、绝望、不甘），最后是意识模糊
- 记忆融合完成后，主角确认两件事：①自己确实穿越到了《剑道天下》的世界；②原主已经死亡（或被取代），自己继承了他的身体和部分记忆
- **结尾钩子**：脑海中出现陌生记忆和不属于自己的人生经历（悬念型），让读者好奇原主的真正身份、为何被袭击、黑风寨与原主有何关联

## 结尾钩子

- **钩子类型**：悬念
- **具体内容**：记忆融合完成后，主角在脑海中看到了原主最后一段模糊记忆——后山深处，一个神秘的石门，石门上刻着与手背剑纹相同的图案；石门缓缓开启，里面有微弱的光芒闪烁，一个模糊的声音似乎在呼唤"……回来……"
- **预期效果**：读者好奇石门是什么、手背剑纹与石门的关联、那个呼唤的声音是谁，迫切想继续阅读下一章
