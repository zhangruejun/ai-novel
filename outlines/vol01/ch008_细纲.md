## 场景拆分

### 场景1：村口血火

| 项目 | 内容 |
|------|------|
| 场景类型 | **B（动作/战斗）** |
| 时间地点 | 中午，十里铺村村口 |
| 出场角色 | 李长风、山贼头目（疤脸）、山贼若干、村民 |
| 核心动作 | 李长风跑到村口，看到十几名山贼正在村里肆虐。房屋起火，村民四处奔逃。一个脸上有刀疤的头目正抓着赵老汉的衣领，逼问"东西在哪"。李长风认出山贼的服饰和旗号——黑风寨。他想起原主记忆中的画面：黑风寨探子深夜在茅屋外喊"那小子死了没"。F010伏笔回收。 |
| 对话要点 | - 疤脸头目："老东西，把东西交出来！"  - 赵老汉咬着牙不吭声  - 疤脸："不说？那就别怪我——"  - 李长风："放开他。"  - 疤脸转头看他，咧嘴笑了："又一个送死的。"  - 疤脸对手下："去，把他也拿下。" |
| 过渡方式 | 两个山贼朝李长风扑来，战斗开始。 |
| 场景细节要求 | 1. 村口的混乱要具体：起火的茅草屋、四散奔逃的村民、倒在地上的人 2. 黑风寨山贼的服饰：统一的黑色短打，左臂缠黑布 3. 疤脸头目的形象：满脸横肉，左脸一道刀疤从眼角延伸到嘴角 4. 李长风认出黑风寨时的心理活动：不是恐惧，而是"终于来了"的冷静 5. ⚠️ F010伏笔回收要自然，通过原主记忆闪回 6. ⚠️ 不要让李长风一上来就碾压，要先展示危机感 |

### 场景2：反杀头目

| 项目 | 内容 |
|------|------|
| 场景类型 | **B（动作/战斗）** |
| 时间地点 | 中午，十里铺村村口到村内 |
| 出场角色 | 李长风、疤脸头目、山贼若干 |
| 核心动作 | 两个山贼扑来。李长风拔出无名铁剑，利用游戏经验分析他们的行动模式。山贼的站位、配合方式、攻击节奏，与游戏中黑风寨小怪的AI行为高度相似。他利用地形优势（村口的老槐树、一口水井、倒塌的院墙）逐个击破两个小喽啰。然后与疤脸头目交手。头目比普通山贼强得多，不是游戏小怪的水平。李长风陷入苦战，左臂伤口崩裂，身上又添新伤。但他发现头目有一个习惯：每次劈砍后有一个微小的停顿。他利用这个破绽，以伤换命，铁剑刺穿头目咽喉。 |
| 对话要点 | （战斗中的内心独白为主）  - "站位跟游戏里一样，呈扇形包抄。"  - "左路的先攻，右路补位。"  - "头目不一样。比游戏里强。"  - "他每次劈完刀，右肩会沉半寸。"  - "就是现在。"  - 头目临死前："黑风寨……不会放过……" |
| 过渡方式 | 头目倒下，剩余山贼见势不妙，纷纷逃窜。 |
| 场景细节要求 | 1. 两个小喽啰的战斗要快速、利落，展示游戏经验的优势 2. 头目的战斗要认真写，不是碾压，是苦战 3. 头目的实力：相当于引气入体巅峰，接近正式修炼者 4. 李长风的胜利来自分析+时机把握，不是实力碾压 5. ⚠️ 战斗代价要大：左臂伤口崩裂，右腿被刀划开，体力接近透支 6. ⚠️ 铁剑的描写：普通但有分量，战斗中剑刃出现缺口 7. 真气的使用要具体但不多，真气不足以支撑长时间战斗 8. 头目的刀法要具体：劈、砍、扫、撩，有套路可分析 |

## 细节要求

### 前文物品提及
- **铜柄断剑**：腰间，柄上古篆"云"字，本章未使用（太短不适合作战）
- **无名铁剑**：主战武器，战斗中剑刃出现缺口
- **军中打法伤药**：林婉儿留下的，战后可以使用（展示疗伤）

### 伏笔处理
- **F010-原主被打伤事件**：本章回收。通过赵老汉的叙述，确认打伤原主的是黑风寨探子
- **F008-铺垫（村民行为异常）**：暂时搁置。本章村民表现正常，被山贼袭击时的反应是真实的
- **F001（系统异常）**：本章面板正常弹出，暂时不异常

### 角色状态提醒
- **李长风**：引气入体，真气初生。战斗后体力透支，多处受伤
- **赵老汉**：被山贼威胁，战后说出部分真相
- **疤脸头目**：黑风寨小头目，被李长风反杀

### 战斗信息
- **对手1**：黑风寨喽啰×2（游戏小怪级别，AI行为可预测）
- **对手2**：疤脸头目（引气入体巅峰，有实战经验，不是游戏小怪水平）
- **战斗方式**：利用地形+游戏经验分析+铁剑+真气引导
- **战斗代价**：左臂伤口崩裂，右腿刀伤，体力透支，铁剑缺损

### 游戏知识的边界
- 山贼喽啰的行为模式与游戏一致（站位、配合节奏）
- 头目与游戏不同：更强、更有章法，不是固定AI
- 主角发现"游戏经验有用，但不能完全依赖"

## 特殊要求

1. **爽点设计**：反杀头目是本章核心爽点。要让读者感受到"主角用智慧战胜强敌"的快感，不是碾压，是有策略的胜利。
2. **F010伏笔回收**：通过赵老汉的叙述自然回收，不要生硬。确认原主被打伤与黑风寨有关。
3. **战斗节奏**：场景1展示危机→场景2小喽快速解决→头目苦战→场景3战后余波。节奏要快慢结合。
4. **铁剑的描写**：战斗中要体现铁剑的"普通"——剑刃会出现缺口，不是神兵利器。
5. **村民的反应**：不要写成跪地感谢，要朴素的、真实的村民反应。有人感激，有人害怕，有人沉默。
