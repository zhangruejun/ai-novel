### 场景一：夜色行路（约900-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | D感情培养 |
| 时间地点 | Day 6 深夜，瘴谷外围通往青州镇的山路上，月色微弱 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 李长风右腿伤加重，林婉儿提出停下换药。两人在路边一块青石旁坐下，林婉儿打开医箱处理伤口 |
| 换药细节 | 林婉儿解开旧布条——血已经凝固，与伤口粘连。她用温水浸湿布条一角，慢慢软化凝固的血块。李长风低头看着她的动作，没有说话。布条完全取下后，伤口暴露在夜风中——皮肉翻卷，边缘发白，有发炎迹象。林婉儿从医箱取出金疮药，均匀撒在伤口上，用干净布条重新包扎。她的动作熟练，手指稳定，没有多余的动作。包扎完成后，她说："右腿不能用力。天亮前找地方休息。"李长风点头 |
| 对话要点 | 包扎过程中，两人讨论瘴气脉动。李长风："脉动从三息缩短到不到两息。"林婉儿："它在加速。如果瘴气是被控制的，控制者在加大输出。"李长风："异兽不攻击我们，更像是在确认。"林婉儿："确认我们是否构成威胁。或者说——确认我们是否知道得太多。"她没有说"控制者"三个字，但意思已经明确 |
| 李长风的决策 | 换药完成后，李长风表态："先去青州镇。休整、疗伤。然后打探归元宗的消息。"林婉儿："镇上有药铺，我的医箱需要补充。"她没有追问归元宗的事，也没有提玉简和铜牌 |
| 过渡方式 | 两人继续前行。夜色加深，山路难行，李长风用无名铁剑拄地支撑右腿。林婉儿走在他身侧，右手始终缩在袖中。前方出现一处背风的山坳，适合休息 |

### 场景二：途中露宿（约900-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | A对话主导（含C悬疑推理元素） |
| 时间地点 | Day 6 深夜→Day 7 凌晨，山路边背风山坳，月色透过树隙照下 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人在山坳中生起小火堆休息。李长风取出玉简，在月光下观察裂纹。裂纹中的灰绿色光芒不再像瘴谷中那样强烈，但仍在以极慢的频率闪烁——与远处隐约传来的瘴气脉动同步 |
| 玉简观察 | 李长风将玉简平放在掌心，裂纹在月光下清晰可见。光芒的闪烁频率比之前在瘴谷中慢了很多——约十息一次。距离瘴谷越远，光芒越弱，但没有完全消失。这说明玉简与瘴眼的联系是持续的，不受距离完全切断 |
| 林婉儿的暗示 | 火光中，林婉儿第一次主动提及与线索相关的内容。她说："云崖子这个名字——我见过。"李长风看向她。她盯着火堆，没有抬头："在一本旧医书上。不是正式记载，是书页边缘的批注。"她没有说医书的名字，也没有说在哪里看到的。李长风没有追问。他记在心里 |
| 对话要点 | 李长风："什么批注？"林婉儿："只有一句话——'瘴非天降，封印有期'。"她停顿，"和玉简里的信息对得上。"李长风："封印有期。意思是封印有期限。"林婉儿点头。她说："如果封印是被设定好的期限，那到期之后——瘴气会全部释放出来。"李长风："所以脉动在加速。封印在彻底失效前，瘴气的释放越来越不受控制。" |
| 剑纹状态 | 对话中，林婉儿的右手在袖中微微颤抖。手背上的剑纹在火光中呈现极浅的银灰色，比夜色中的玉简光芒更淡。她将手攥成拳，然后松开，重复了两次。李长风看到了这个动作，但没有询问 |
| 过渡方式 | 远处传来脚步声——不是一个人的脚步，而是有节奏的、整齐的步伐，像是队伍行进。声音来自前方，青州镇方向。两人同时停止交谈，看向声音传来的方向。脚步声持续了约半柱香后消失 |

### 场景三：青州镇在望（约900-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境氛围（含悬疑元素） |
| 时间地点 | Day 7 天亮，青州镇外围山坡，可俯瞰全镇 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 天亮后两人继续前行，登上镇外一处山坡，从远处观察青州镇 |
| 青州镇外观 | 青州镇比预期冷清。镇墙是夯土筑成，高约两丈，墙头有巡逻步道。镇门大开，但进出的人稀少——只有两辆牛车和三五个行人。镇内街道 visible，但行人稀疏，不如正常集镇的热闹。几家店铺的招牌还在，但门面紧闭 |
| 归元宗痕迹 | 李长风用从猎户处借来的旧望远镜（ch011获得）观察镇口——镇门外左侧有一处废弃营地：灰烬未冷（昨夜或凌晨离开），地面有靴印（军用制式），还有几根插过旗帜的木桩。木桩已经拔走，但木桩周围的泥土被翻动过。他从山坡下到营地旁，仔细查看——靴印尺寸统一，步幅一致，是训练有素的人留下的。营地呈方形布局，符合宗门野外驻扎的规制 |
| 确认发现 | 李长风回到山坡上，说："归元宗在青州镇有活动。"林婉儿："驻扎过，但已经走了。"李长风："昨天夜里。或者今天凌晨。"他判断："不是长期驻扎，是临时行动。" |
| 环境细节 | 镇内方向传来钟声——不是寻常的晨钟。寻常晨钟是三声，间隔均匀。这钟声响了五声，间隔不规则，第一声和最后一声特别沉重，中间三声急促。钟声传出不远，像是被什么阻挡了 |
| 结尾钩子 | 李长风记下钟声的规律。他说："不是寻常钟声。"林婉儿侧耳听完，说："是召集令。但不是召集镇民——是召集自己人。"她补充："我听过这种钟声。"李长风看向她。她没有解释在哪里听过。两人沿山坡下行，朝镇门方向走去 |
| 过渡方式 | 两人走近镇门。镇门口的守卫穿着青州镇本地制服，但眼神警惕，手按刀柄。镇内街道上能看到几处归元宗弟子留下的标记——墙角的云纹符号、路灯杆上的布条。下一章将在青州镇内展开 |

## 结尾钩子

- **钩子类型**：悬疑+新线索
- **具体内容**：青州镇比预期冷清，镇外有归元宗临时驻扎营地（昨夜或凌晨离开），镇内传来不寻常的五声钟声（召集令，非召集镇民）。林婉儿暗示听过这种钟声但不肯多说
- **预期效果**：读者对归元宗在青州镇的活动目的产生好奇，对异常钟声的含义感到紧张，想要知道两人进入青州镇后会发现什么

## 特殊要求

- 换药场景要写实，展现林婉儿的医理能力，但不要写得太温柔——两人关系是"互相信任的同伴"，不是暧昧
- 对话简短有力，避免长篇大论。每人说话不超过三句，用停顿和动作补充信息
- 玉简观察要体现距离与光芒强度的关系——离瘴谷越远光芒越弱，但不消失
- 林婉儿对"云崖子"的暗示要克制——她提供了线索（旧医书批注），但拒绝深入。李长风不追问
- 脚步声场景要有紧张感——两人同时停止交谈，看向声音方向，手按武器
- 青州镇外观描写要体现"冷清"——用细节（紧闭的店门、稀少的行人、空旷的街道）而非直接说"冷清"
- 归元宗营地痕迹要专业——靴印尺寸统一、步幅一致、方形布局，体现李长风的观察力和判断力
- 钟声描写要有节奏感——五声，间隔不规则，第一声和最后一声沉重，中间三声急促
- 林婉儿对钟声的反应要自然——"我听过这种钟声"，但不解释。让读者好奇
- 场景之间过渡自然：行路换药→山坳休息讨论→天亮到达青州镇
- 不要提前揭露林婉儿的穿越者身份
- 禁用词检查：仿佛、似乎、感觉到、意识到、不知为何、莫名、一股、一丝、缓缓、渐渐、然而、与此同时、值得注意的是、不得不提、众所周知、毫无疑问、显而易见、毋庸置疑、不言而喻、可以说、换句话说、也就是说、总而言之、不仅如此、更重要的是、除此之外、反观、诚然、无独有偶、犹如、宛如、如同
