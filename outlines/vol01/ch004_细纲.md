## 本章在卷中的位置

- **承上**：承接第003章结尾，主角出门前往后山矿洞，三天倒计时已开始
- **启下**：为第005章系统异常加剧做铺垫——断剑上的古篆字系统无法识别，暗示系统与这个世界存在某种"不兼容"
- **本章作用**：推进剧情（获得第一把剑）、提升实力（真气觉醒）、埋设伏笔（F010-1古篆字"云"）

## 本章核心

- **核心事件**：主角前往后山矿洞探索，在矿洞深处发现枯骨并获得铜柄断剑，断剑入手瞬间体内真气觉醒
- **情绪基调**：紧张→震惊→兴奋
- **核心冲突**：主角面对未知矿洞环境的恐惧与获得力量的渴望之间的内心冲突

## 前章衔接

- **上一章结尾**：第003章结尾，主角拔开门闩，拉开木门，迈步跨出门槛，倒计时开始
- **本章开头**：主角离开茅草屋，沿着村后小路向后山矿洞方向走去，日头偏斜，田野间弥漫着傍晚的气息
- **需要注意的连续性**：
  - 时间流逝：第003章结尾日头过了正午，本章开头应为傍晚时分
  - 身体状态：左肩钝痛还在，恢复中，但不影响行动
  - 系统面板已激活，任务栏亮着【新手任务一：在三天内获得第一把剑】
  - 林婉儿处于退场状态（第2-11章不出现），本章不提及

## 场景拆分

### 场景一：矿洞入口（约900字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E（环境氛围） |
| 时间地点 | 傍晚，后山矿洞入口 |
| 出场角色 | 李长风 |
| 核心动作 | 主角找到矿洞入口，观察周围环境，感受到矿洞的阴森气息，犹豫后决定进入 |
| 对话要点 | 无对话，内心独白为主——分析游戏里矿洞的布局、评估当前风险、回忆原主对矿洞的模糊记忆 |
| 过渡方式 | 主角点燃随身携带的火折子，踏入矿洞，光线骤暗，自然过渡到场景二 |

**详细要求**：
- 环境描写为主：矿洞入口被杂草半掩，石块堆积，隐约可见向下的阶梯
- 氛围营造：傍晚的风带着凉意，远处有乌鸦叫声，矿洞口散发着潮湿泥土和腐朽木材的气味
- 主角心理：游戏经验告诉他矿洞里有废弃工具和断剑，但现实中的矿洞比游戏更真实、更危险
- 细节：矿洞口有几道深深的爪痕（不是动物，像是什么利器划过），暗示矿洞曾有人活动

### 场景二：矿洞深处（约1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | B（动作场景） |
| 时间地点 | 夜晚，矿洞深处（废弃矿道） |
| 出场角色 | 李长风 |
| 核心动作 | 主角在矿洞中摸索前进，发现一具枯骨，枯骨旁有一把铜柄断剑，主角上前取剑 |
| 对话要点 | 无对话，内心独白——分析枯骨的来历（老矿工？）、对断剑品质的判断、取剑前的谨慎检查 |
| 过渡方式 | 手指触碰到断剑的瞬间，异变突生，过渡到场景三 |

**详细要求**：
- 动作描写：主角小心翼翼地沿着矿道前进，脚下是碎石和腐朽的枕木，火折子的光摇曳不定
- 发现枯骨：在矿道岔口处，一具枯骨靠在石壁上，衣衫褴褛，骨骼呈灰白色，死了至少十年
- 枯骨细节：枯骨右手伸向前方，似乎在生命的最后一刻想要抓住什么；左手边地面上有一把剑
- 断剑描写：铜柄，剑身从中间断裂，只剩约两尺长，锈迹斑斑但剑柄处隐约可见暗纹
- 取剑过程：主角蹲下，用树枝拨弄断剑确认无陷阱，然后伸手握住剑柄
- 紧张感：矿洞深处偶尔传来"滴答"的水声，火折子的火苗突然变小，空气变得沉闷

### 场景三：真气觉醒（约1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C（悬疑） |
| 时间地点 | 夜晚，矿洞深处（枯骨旁） |
| 出场角色 | 李长风 |
| 核心动作 | 断剑入手瞬间体内真气觉醒，断剑柄上古篆字"云"闪过，主角联想到原主记忆碎片，震惊于自身变化 |
| 对话要点 | 内心独白——真气运行的感受、对古篆字的疑惑、对断剑来历的猜测 |
| 过渡方式 | 主角握着断剑站起身，感受到体内流动的真气，矿洞外传来隐约的风声，本章结束，留下悬念 |

**详细要求**：
- 真气觉醒描写（重点）：
  - 手指握住剑柄的瞬间，一股热流从掌心窜入，沿着手臂经脉向上蔓延
  - 经脉传来轻微的胀感，像干涸的河道突然涌入溪水
  - 热流最终汇聚到丹田，形成一个温热的气团
  - 身体表面的汗毛微微竖起，耳边出现低沉的"嗡嗡"声
  - 主角能清晰感知到体内这股前所未有的力量在流动
- 伏笔埋设F010-1（"云"字记忆碎片）：
  - 断剑剑柄处有一个古篆字，笔画古朴
  - 主角辨认出那个字似乎是"云"
  - 看到"云"字的瞬间，脑海中闪过一段不属于原主的记忆碎片：一个模糊的身影，一柄长剑，一句低语"云……散……"
  - 记忆碎片一闪即逝，主角无法捕捉更多
  - 主角下意识调出系统面板，试图识别那个字，但系统显示"无法识别"或"字符编码错误"（与第003章乱码呼应）
  - 主角联想到原主记忆中曾听人提起过"云"字相关的势力或人物，但记忆太模糊，无法确定
- 主角反应：
  - 震惊：这是武道修炼的第一步——引气入体，感应真气
  - 兴奋：终于有了自保之力，任务完成在即
  - 疑惑：断剑上的古篆字是什么？那段记忆碎片是谁的？为什么系统无法识别？
- 结尾钩子设计：
  - 断剑入手瞬间体内真气觉醒
  - 主角发现自己能感应到断剑上残留的某种气息（不属于这个世界的气息）
  - 悬念型：那气息冰冷而古老，仿佛跨越了漫长的岁月，正在"注视"着他

## 结尾钩子

- **钩子类型**：悬念型（重大发现+新力量觉醒）
- **具体内容**：主角握住断剑的瞬间，体内真气觉醒，但同时发现断剑上残留着某种不属于这个世界的古老气息，系统无法识别断剑上的古篆字"云"，脑海中闪过一段模糊的记忆碎片
- **预期效果**：读者对断剑的来历产生好奇，对古篆字"云"的含义产生疑问，期待后续揭示

- 《剑道天下》游戏经验（主角用来判断矿洞布局和断剑位置）

### 伏笔处理

- **需要呼应的伏笔**：F001a（系统乱码现象）——系统尝试识别古篆字时出现"无法识别"或"编码错误"，与第003章的乱码呼应

- 林婉儿：退场中（第2-11章不出现），本章绝不提及
- 主角情绪变化：紧张（矿洞探索）→震惊（真气觉醒）→兴奋（获得力量）→疑惑（断剑之谜）

### 战斗信息
- 本章无战斗

### 写作约束
- **真气觉醒描写要求**：必须包含身体感受的具体描写——热流涌动、经脉微胀、丹田温热等，让读者能"感受到"真气的存在
- **古篆字"云"埋设要求**：自然融入剧情，不刻意点明，不展开解释，只作为悬念留下
- **林婉儿约束**：本章不出现、不提及、不暗示
- **节奏控制**：场景一（环境）→场景二（动作/发现）→场景三（悬疑/觉醒），情绪从紧张到震惊到兴奋，逐步递进
- **字数分配**：三个场景总字数控制在2800-3200字

## Prompt模板选择

- **主要使用模板**：基础模板（场景一、三）+ 动作模板（场景二）
- **辅助使用模板**：悬疑模板（场景三的伏笔埋设部分）

## 特殊要求

1. **环境描写克制**：场景一的环境描写控制在3-5句内，避免过度渲染恐怖氛围
2. **真气觉醒真实感**：用具体的身体感受替代抽象的"力量涌来"，让读者能感知到真气的存在
3. **古篆字处理**：不直接告诉读者"云"字代表什么，只展示主角看到的、想到的、疑惑的
4. **系统识别失败**：与第003章的乱码呼应，暗示系统在这个世界存在"兼容性"问题
5. **断剑品质**：断剑不是神兵利器，只是一把普通的废弃断剑，但剑柄处的古篆字暗示它来历不凡
