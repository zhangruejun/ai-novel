## 场景拆分

### 场景一：城门震撼（约800字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境 |
| 时间地点 | 九月十六日下午，归元城城门 |
| 出场角色 | 李长风、张云义 |
| 核心动作 | 走近归元城，两人都被震撼。

**【归元宗地位烘托·顶流写法：用细节烘托，不靠嘴说】】**
城墙高达十丈，由整块青石砌成，上面刻满了防御阵法符文——这种规模的城防阵法，主角只在游戏里的皇都见过。城门高达三丈，上方悬挂一块巨大的黑底金字匾额，上书"归元城"三个大字，笔力雄浑，字中蕴含的剑意让主角心中一震——这是剑道大能亲手题字。

城门两侧各站着八名归元宗外门弟子（气息都在剑师境以上），以及八名朝廷城防军士兵——比平时多了一倍人手。主角注意到一个细节：城防军站在外侧，归元宗弟子站在内侧——在这座城里，归元宗才是真正的主人，朝廷反而像是配合者。

城墙上的阵法符文中心，有一个巨大的黑色莲花浮雕——归元宗的宗门徽记，但莲花的造型让主角心中微动（他已经在杀手令牌和密信上见过这个标记了）。

**【考核期拥挤感·顶流写法：用细节烘托人气】】**
进城分两条通道：左边通道对普通人免费，右边武者通道每人收取五文铜钱（朝廷规定的武者入城费）。修士凭宗门令牌免收。但今天武者通道排了足足百人的长队——全是来参加外门考核的考生和护送他们的随从。主角排了半个时辰才进城。

城内景象更震撼——宽阔的青石大道可同时容纳十辆马车并行，但今天街上挤满了人，各种口音的武者、商人、修士混杂其中。客栈街方向传来掌柜的吆喝声："客满！客满！各位仙师去别家看看吧！"——外门考核在即，归元城客栈已经爆满。主角注意到城中随处可见的归元宗巡逻队，每队都有一名剑宗境带队——这种配置，堪比一些小宗门的全部家底了 |
| 对话要点 | 张云义倒吸一口凉气："这就是归元城......"主角没有说话，但心中同样震撼 |
| 爽点/钩子 | 城市规模震撼，世界观扩展，朝廷痕迹细节 |
| 过渡方式 | 两人进城找客栈 |

### 场景二：找客栈住下（约1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | A对话/E环境 |
| 时间地点 | 九月十六日下午到夜间，归元城内 |
| 出场角色 | 李长风、张云义、客栈掌柜、其他找房的武者 |
| 核心动作 | **【考核期客栈爆满·顶流写法：用找房困难烘托人气】】**
两人在城内找客栈，连问三家都说"客满"。第四家"云来客栈"的掌柜是个精明的中年胖子，看两人穿着朴素但气质不凡（武者气息），勉强安排了一个二楼的独立小院——本来是给叶家留的，但叶家包了整条街的客栈。住宿费：平时一天一两银子，考核期间一天三两银子。主角付了五天的钱（十五两）。

掌柜主动搭话："两位仙师是来参加外门考核的吧？今年春季考核听说有四百多人报名呢，连城外十里地的客栈都满了。"

住下后，主角把重要证据（归元宗长老令牌、黑色莲花令牌、通信）藏在一个隐蔽的墙缝里，只随身携带必要物品。

张云义："报名处就在城北归元宗山门外。"
主角："今天先休息，明天一早去报名。报名期据说有五天，不急。" |
| 对话要点 | 客栈掌柜："两位仙师是来参加外门考核的吧？"张云义："外门考核明天开始。"主角："今天先休息。" |
| 爽点/钩子 | 安全落脚点建立 |

### 场景三：打听考核信息（约800字）

| 项目 | 内容 |
|------|------|
| 场景类型 | A对话 |
| 时间地点 | 九月十六日夜间，客栈内 |
| 出场角色 | 李长风、酒楼食客们 |
| 核心动作 | 主角没有出门（客栈爆满街上人多，不惹眼），在客栈房间里和张云义整理明天的计划。张云义从楼下打茶回来，带来了街上听到的消息：①外门考核每半年一次，这次是春季考核，报名参加的有四百多人；②报名期从第9天到第13天，共五天，让各地考生有时间赶到；③报名后还要等三到五天才正式考核；④考核分三关：第一关测骨龄和境界（骨龄不超过二十岁，境界不低于剑徒境）；⑤第二关是实战考核，在试炼谷中猎杀妖兽；⑥第三关是心性测试，由归元宗长老亲自面试。

张云义还提到："街上看到好几拨人了。青州苏家的车队刚过去，铁剑山庄的人也来了。明天报名肯定热闹。"

主角心里有数了。报名期五天，报名后还要等考核——这才是大宗门的做派。第一关和第二关没问题，第三关心性测试需要小心——王长老可能在里面做手脚 |
| 对话要点 | 张云义："报名期五天，考核还要等。"主角："大宗门的做派。" |
| 爽点/钩子 | 考核信息获取，王长老监考暗示 |
| 过渡方式 | 主角回客栈与张云义汇合 |
