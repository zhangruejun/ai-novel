## 场景拆分

### 场景一：沿途观察（约1500-1600字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑 |
| 时间地点 | Day 5 清晨→上午，十里铺村外的土路→路旁村庄 |
| 出场角色 | 李长风（独自主视角） |
| 核心动作 | 李长风沿途观察其他村庄的正常农事，与十里铺村的异常行为对比，回想并点出F008-动作伏笔 |
| 对话要点 | 无对话，主角内心独白为主 |
| 过渡方式 | 观察结束后继续赶路，在路边休息时林婉儿出现 |

**场景细节要求**：
1. 李长风走了一炷香后，路过一个正常村庄，看见村妇打水时弯腰喘息、农夫除草时动作随意且不时停下来擦汗
2. 对比十里铺村：ch002中村妇打水全程无抬头无喘息、ch002中除草者节奏稳得不像干活——此刻回想点出异常
3. 李长风内心分析：十里铺村民不是在做事，是在执行固定动作序列；其他村民有正常人的疲惫和随意
4. 主角注意到路边农田：正常村庄的田里庄稼长势良好，偶尔有杂草，农夫会停下来休息喝水
5. 回忆赵老汉"收成差了三成"——十里铺村民白天确实在"劳作"但都是固定动作，不是真种地，产量暴跌是必然的
6. 李长风得出结论但暂不深究：这个村子有问题，但不是他能立刻解决的；先活下去，再查真相
7. 情绪从悬疑压抑转为释然：至少离开了那个地方

### 场景二：林婉儿出现（约1300-1600字）

| 项目 | 内容 |
|------|------|
| 场景类型 | E环境氛围 |
| 时间地点 | Day 5 上午，路边休息处（一棵老柳树下） |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 林婉儿在路边出现，主动要求同行，给出看似合理的理由，李长风警惕但暂不拒绝 |
| 对话要点 | 林婉儿说明来意（路过/听说黑风寨不太平/一个人走不安全），李长风试探性回应，双方保持礼貌但暗中观察 |
| 过渡方式 | 两人达成暂时同行的默契，继续上路，结尾钩子落在林婉儿的动机不明 |

**场景细节要求**：
1. 环境描写：初春阳光照在老柳树上，枝条抽出新芽，地上有斑驳的光影；路边有野花开放，远处传来鸟鸣
2. 林婉儿出场方式：不是突然出现，而是从路的那头走来，背着一个小包袱，穿着素色布衣，看起来像赶路的普通女子
3. 李长风第一反应是警惕，手按上剑柄，但没有拔剑——观察她的举止
4. 林婉儿的表现：呼吸微促（看起来真的走了一段路），额头有细汗，但眼神清明，步伐稳定——与"疲惫赶路"的状态有微妙矛盾
5. 对话节奏：林婉儿主动开口，语气自然但每句话都有目的性；李长风回应简短，多用问句试探
6. 林婉儿给出的同行理由："我要去前面的镇子，一个人走不太安全。听说你要去黑风寨方向，顺路。"——理由合理但经不起推敲
7. 结尾钩子：李长风注意到林婉儿手背上有一道淡淡的剑纹痕迹（与自己的剑纹相似但不完全一样），她很快将手缩回袖中

## 结尾钩子

- **具体内容**：李长风在对话中注意到林婉儿手背上有一道淡淡的剑纹痕迹，与自己的剑纹相似但不完全一样。林婉儿很快将手缩回袖中，转移了话题。
