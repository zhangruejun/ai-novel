| 项目 | 内容 |
|------|------|
| 场景类型 | **A（对话主导）** |
| 时间地点 | 早晨，十里铺村村口 |

| 核心动作 | 村长敲响铜锣，发布任务：后山出现一头野猪，已经拱坏了三亩玉米地，需要有人去清理。奖励：无名铁剑一柄（从过路商人处换来）。李长风主动接下任务，村民无人响应。 |
| 对话要点 | - 村长描述野猪的破坏情况  - 李长风主动接下任务，村民们议论纷纷  - 有村民提醒："那畜生凶得很，前年伤了两个人"  - 李长风简短回应，不废话  - 村长把剑交给他，嘱咐当心 |

### 场景2：后山野猪

| 项目 | 内容 |
|------|------|
| 场景类型 | **B（动作/小规模战斗）** |
| 时间地点 | 上午，十里铺村后山 |
| 出场角色 | 李长风、野猪 |
| 核心动作 | 李长风来到后山，利用游戏知识找到野猪的刷新点。野猪出现（体型大、獠牙锋利）。李长风不急于动手，先观察野猪的行为模式，确认与游戏中的AI行为一致。然后利用地形优势（一棵斜长的树、一块凸起的岩石）和真气引导，轻松击杀野猪。 |
| 对话要点 | （战斗中的内心独白为主）  - "位置没错。"  - "跟游戏里一样，刷新后会在原地巡逻三圈。"  - "弱点在左肋下三寸。"  - "真气走手少阳三焦经，快。"  - 战斗结束后："引气入体，杀一头野猪足够了。" |
| 过渡方式 | 李长风割下野猪的一根獠牙作为凭证，带着断剑（剑身太短，不适合作战）和缴获的獠牙回村。 |
| 场景细节要求 | 1. 野猪的战斗要具体：獠牙的威胁、冲锋的速度、李长风的闪避动作 2. 李长风使用断剑+真气，不是蛮力 3. 利用地形是游戏玩家的标志性行为 4. ⚠️ 不要写成碾压，要写成"有策略的胜利" 5. ⚠️ 战斗要有代价：李长风的衣服被獠牙划破，左臂擦伤 6. 真气的使用要具体：从丹田引导，走特定经脉，到达掌心 |

## 细节要求

### 前文物品提及
- **铜柄断剑**：李长风的主武器，但剑身太短，不适合作战。柄上古篆"云"字。
- **无名铁剑**：任务奖励，普通但完整的一柄铁剑。
- **军中打法伤药**：林婉儿留下的，李长风可以使用（但本章不需要用）。

### 伏笔处理
- **F008-铺垫（村民行为异常）**：暂时搁置。本章村民表现正常，与第6章形成对比（说明异常只在夜间出现）。
- **F001（系统异常）**：暂时平息。面板在任务完成时正常弹出，让读者放松警惕，为后续异常加剧做反差。

### 角色状态提醒
- **李长风**：引气入体，真气初生，左肩伤势大幅减轻。手持铜柄断剑。昨夜未眠但精神尚可。
- **村长（赵老汉）**：普通NPC型村民，本章表现正常。

### 战斗信息
- **对手**：后山野猪（野兽，非人类）
- **战斗方式**：利用地形优势+真气引导+断剑刺击
- **战斗代价**：衣服被划破，左臂轻微擦伤

## 特殊要求

1. **爽点设计**：本章是第一个"爽点"章节，要让读者感受到主角利用游戏知识带来的优势。但不要写成碾压，要写成"有智慧的胜利"。
2. **游戏知识的边界**：主角发现这个世界和游戏"高度相似"但不完全一样。野猪的行为模式一致，但细节有差异（更真实、更危险）。
3. **村民恢复正常**：第6章的异常只在夜间出现，白天村民表现正常。这是为了让读者和主角一样产生"也许是我多想了"的错觉。
4. **无名铁剑的描写**：这柄剑很普通，不是神兵利器。但它是一把"完整的剑"，与断剑形成对比。
5. **系统正常的反差**：面板在任务完成时正常弹出，没有乱码。这是为了让读者放松警惕，后续异常加剧时反差更大。
