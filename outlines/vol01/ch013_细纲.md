## 本章在卷中的位置

- **承上**：承接第012章李长风与青衣帮交手后首次意识到"世界的人是真实的"，两人接近青州镇
- **启下**：为第014章探索危险区域获得宗门招收弟子情报做铺垫
- **本章作用**：推进剧情+塑造角色（展现李长风的正义感和底线）+感情培养（与李长风、林婉儿之间的信任建立）

## 本章核心

- **核心事件**：李长风与林婉儿在接近青州镇的路上遭遇一伙地痞勒索过路书生，李长风出手相助
- **情绪基调**：热血→满足
- **核心冲突**：地痞以"问路钱"为名勒索书生，书生无力反抗，李长风选择介入

## 前章衔接

- **上一章结尾**：李长风打退青衣帮头目后与林婉儿继续前行，远方出现青州镇轮廓，林婉儿说"快到了"。李长风首次意识到世界的人是真实的，不是NPC
- **本章开头**：两人继续朝青州镇方向走，日头偏西，路旁出现茶摊。李长风伤势仍在恢复中，左臂动作僵硬，右腿隐隐作痛
- **需要注意的连续性**：①李长风左臂和右腿伤势未愈，战斗中需体现限制；②干粮剩余2日量，水囊尚满；③林婉儿始终表现平静，对江湖纠纷见怪不怪；④两人感情温度15/100，处于互相试探建立信任阶段

## 场景拆分

### 场景一：茶摊见闻（约900-1000字）

| 项目 | 内容 |
|------|------|
| 场景类型 | D感情培养 |
| 时间地点 | 下午，青州镇外三里处路边茶摊 |
| 出场角色 | 李长风、林婉儿、茶摊老翁（60岁左右，沉默寡言）、过路书生（20出头，青衫，背书箱）、地痞三人（头目40岁，瘦高，两个跟班） |
| 核心动作 | 两人在茶摊歇脚，目睹地痞勒索书生。书生被推倒在地，书箱翻倒，竹简散落一地。地痞头目踩住竹简，索要"问路钱"。李长风观察局势，林婉儿表现平静 |
| 对话要点 | ①李长风与林婉儿就"该不该管"有简短交锋——林婉儿问"你打算怎么做"，不直接表态；李长风沉默后说"看不惯"；②茶摊老翁低头倒茶，装作没看见；③地痞头目与书生的对话——头目语气慢条斯理但带着威胁，书生紧张但不卑不亢 |
| 过渡方式 | 地痞头目发现李长风在盯他，目光对视，气氛紧张，自然过渡到冲突 |
| 场景细节要求 | ①茶摊简陋：四张方桌，长凳有裂纹，茶是大叶粗茶，两文一碗；②书箱翻倒后竹简散落，有几卷滚到李长风脚边，他弯腰拾起，竹简上写着"归元剑诀"四字；③地痞头目走路时双手背在身后，像官差巡街；④林婉儿喝茶时手指轻轻摩挲杯沿，目光始终没有离开地痞；⑤李长风左臂伤口被牵扯，微微皱眉但没表现出来 |

### 场景三：书生的情报（约800-1000字）

| 时间地点 | 茶摊旁，地痞逃散后 |

| 场景细节要求 | ①书生说话时习惯性地整理衣袖，紧张时会摸鼻子；②提到断魂谷时，茶摊老翁抬头看了一眼，又低头继续干活，像是听到了什么但不想说；③林婉儿问话时语气平静，但目光锐利，像在判断书生话的真假；④李长风听到"归元宗招收弟子"时，手背剑纹微微发热；⑤结尾时太阳西斜，天边有火烧云，远处隐约可见青州镇轮廓更清晰了 |

## 结尾钩子

- **钩子类型**：预告型
- **具体内容**：书生陈文远提到断魂谷近日发生怪异之事——谷中有怪声如人哭泣，多拨人进去后未归。李长风与林婉儿商议后决定前往断魂谷探索
- **预期效果**：读者好奇断魂谷到底发生了什么，是否与穿越/系统/高维文明有关，期待下一章的探索

## 特殊要求

1. **战斗差异化**：本章战斗与第012章青衣帮战斗形成对比。青衣帮头目有章法、有旧伤、会试探；地痞三人粗浅拳脚、无配合、虚张声势。两场战斗风格完全不同
2. **林婉儿的态度**：她始终不主动表态，只用问句引导李长风做决定。"你打算怎么做？""什么人去的？""去看看？"——这是她的一贯风格
3. **禁用AI高频词**：严格检查禁用语列表，特别是"然而""与此同时""嘴角勾起""眼中闪过"等
4. **环境描写**：茶摊环境要具体、生活化，避免"月光如水""阳光透过树叶缝隙"等AI式描写
5. **元叙事禁止**：角色不可能知道"章节"概念，不要用"第X章"作为时间计量
6. **感情推进**：本章感情温度从15提升到18，幅度小但真实。林婉儿对李长风"看不惯就管"的态度有微妙反应，但不表现在脸上
7. **断魂谷预告**：书生的话要自然、有信息量，不能像念设定。用具体细节（怪声、人数、职业）而非抽象描述
8. **场景类型轮换**：本章D→B→A，避免与第012章（A→B）连续重复，与卷规划建议的D→A一致（增加一个B战斗场景）
