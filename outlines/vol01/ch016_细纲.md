### 场景二：发现遗迹（约1100字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑发现 |
| 时间地点 | 石峡通道尽头，断魂谷深处，一片开阔地 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 两人走出石峡，眼前出现一片人类活动遗迹。不是村庄或城池，而是一片残破的石屋群落——约莫七八间石屋散布在瘴气较淡的区域，石屋用青石垒砌，墙壁大半坍塌，屋顶已不存在，只剩下半人高的石墙框架。石屋排列有规律，不是随意搭建，中间有一条石板铺成的小路，路面上长满青苔和一种灰绿色的、像菌丝一样的植物。遗迹中央有一座较大的石屋，明显是核心建筑，比其他石屋高出半丈，门口立着两根断裂的石柱。李长风走进最大石屋，发现屋内地面铺着整块石板，石板中央有一个圆形凹槽，凹槽直径约一尺，深度三寸，槽底刻满密密麻麻的纹路——不是装饰纹样，更像是某种阵法的刻痕。凹槽边缘残留着几块碎裂的黑色石头，石头断面有晶状结构，像是某种灵石 |
| 对话要点 | ①林婉儿走出石峡，环顾四周："有人住过这里。"——她指着石屋排列规律；②李长风检查石屋墙壁："不是住过。"——他手指抚摸石壁上焦黑的痕迹，"是修炼过。"——石壁内部有被高温灼烧过的痕迹，不是普通火烧，更像是真气或灵力冲击留下的；③林婉儿蹲下检查地面菌丝："这些植物……和异兽体内的东西一样。"——她用短刃拨起一丛菌丝，菌丝断面渗出灰绿色雾气；④李长风走进最大石屋，看到圆形凹槽："这是聚灵阵。"——他辨认出槽底纹路，虽然被腐蚀大半，但基本结构还能辨认，是修仙世界中用于聚集天地灵气的阵法；⑤林婉儿："聚灵阵？"——她语气中有疑问，不是惊讶，而是确认；⑥李长风："有人在这里修炼。很多人。"——他从石屋规模和阵法规格推断，这不是一个人的修炼地，而是一个群体的；⑦李长风拿起一块碎裂的黑色灵石碎片，碎片在他掌心微微发热 |
| 过渡方式 | 李长风在聚灵阵凹槽中发现石板侧面有一道暗格缝隙，他用铁剑挑开暗格，里面露出一截被油布包裹的物品 |
| 场景细节要求 | ①石屋群布局：七八间石屋呈环形排列，最大石屋在中央，石板小路连接各屋，整体占地约三十步见方；②石屋材质：青石垒砌，石块之间不用泥浆粘合，而是用某种黑色物质填充，黑色物质已风化剥落；③石壁焦痕：最大石屋内壁从地面到一人高处都有焦黑痕迹，焦痕呈放射状从中央向外扩散，像是有人坐在中央释放大量真气或灵力；④菌丝植物：灰绿色，贴地生长，触感柔软但有韧性，短刃割断后断面渗出雾气，与异兽伤口渗出物相同，暗示异兽与遗迹有直接关联；⑤聚灵阵：圆形凹槽直径一尺，槽底刻纹呈同心圆排列，共九层同心圆，每层有十二个等分点，中心有一个小孔——典型的修仙世界聚灵阵，但规模远超寻常；⑥黑色灵石碎片：共三块，拇指大小，断面有晶状棱面，碎片边缘不规则像是被暴力击碎，李长风拿起碎片时掌心感到微弱温热，类似于手背剑纹的反应但更弱；⑦瘴气浓度变化：遗迹区域瘴气明显变淡，高度只到脚踝，灰绿色变浅，说明聚灵阵仍有残余效果在驱散瘴气；⑧李长风右腿伤势持续影响行动：进入遗迹后每走几步就需调整重心，但他没有停下来处理伤口；⑨林婉儿手背剑纹在遗迹中反应加剧：从温热变为微微发烫，她下意识将手缩入袖中，但没有告诉李长风 |

### 场景三："云"字线索（约900字）

| 项目 | 内容 |
|------|------|
| 场景类型 | C悬疑发现 |
| 时间地点 | 最大石屋内，聚灵阵暗格处 |
| 出场角色 | 李长风、林婉儿 |
| 核心动作 | 李长风用铁剑挑开暗格，发现一截被油布包裹的物品。油布已经风化发脆，一碰就碎，但里面包裹的东西保存相对完好——是一面铜牌。铜牌直径三寸，厚度两分，正面刻着两行字，背面有一个图案。正面文字："云崖子，归元宗内门弟子，入瘴谷探索，不得归"。背面图案：一座山峰，山峰顶部有一朵云的简笔纹。铜牌边缘有一个穿孔，说明原本是挂在脖子上的身份牌。李长风看到"云"字时瞳孔一缩，将铜牌与腰间断刀对比——铜牌上的"云"字，与断刀柄上的"云"字、矿洞断剑上的古篆"云"字，笔画结构完全一致。铜牌下方还有一行小字，被腐蚀了大半，只能辨认出几个字："……瘴非天……人……为……封……" |
| 对话要点 | ①李长风挑开暗格，取出油布包裹："有东西。"——油布一碰就碎，露出铜牌；②林婉儿凑过来看铜牌正面："云崖子……归元宗……"——她念出铜牌上的字，语气平静但语速放慢；③李长风看到"云"字，沉默。他将铜牌翻过来看背面山峰和云纹，然后从腰间取出断刀，将铜牌与断刀并排放在一起；④林婉儿看到两个"云"字："是一样的。"——不是疑问，是确认；⑤李长风："矿洞那把断剑上也有。"——他声音很低，"三个'云'字，同一个来源。"——矿洞枯骨手中的断剑、断魂谷口的断刀、现在铜牌上的"云崖子"，三条线索指向同一个人或势力；⑥林婉儿指着铜牌下方小字："这行字……'瘴非天'，后面是什么？"——只能辨认出这几个字；⑦李长风："瘴非天……人为……封……"——他试着补全，"瘴气不是天然的，是人为封——封锁？封印？封镇？"；⑧林婉儿："人为制造的瘴气？"——她没有用"怎么可能"之类的话，而是直接接受这个推断并继续分析；⑨李长风没有回答。他收起铜牌，环顾石屋，目光落在聚灵阵上——如果瘴气是人为制造的，那这个聚灵阵的作用就不是修炼，而是在对抗瘴气；⑩此时遗迹外围传来异兽的低吼声，但没有靠近——它们畏惧这片区域 |
| 过渡方式 | 李长风将铜牌收好，站起身准备继续探查石屋其他角落。他走到石屋后方，发现后壁有一道裂缝，裂缝中露出半截石阶，通往下方——遗迹还有地下部分 |
| 场景细节要求 | ①铜牌材质：青铜，表面有铜绿，但文字和图案刻痕较深，没有完全被腐蚀；②"云"字比对：铜牌正面"云崖子"的"云"字是标准楷书体，与断刀柄上刻的"云"字笔画结构相同，与矿洞断剑上古篆"云"字结构不同但明显有关联——铜牌楷书、断刀楷书、断剑古篆，三种不同字体但同一个字；③油布包裹：油布已风化发脆，呈灰白色，一碰就碎成粉末，说明年代久远，至少数十年甚至上百年；④铜牌下方小字：腐蚀严重，"瘴非天……人……为……封……"几个字能辨认，其余模糊，字体比正文小一号，像是附加说明；⑤李长风收起铜牌时动作小心，用残存油布碎片重新包裹，放在贴身衣袋中——与断刀、断剑同样对待；⑥林婉儿在李长风比对"云"字时目光一直盯着铜牌，手背剑纹在此刻突然烫了一下，但她没有表现出来；⑦场景结尾时李长风发现地下石阶，裂缝宽约一尺，石阶往下延伸没入黑暗中，台阶上有水渍和菌丝痕迹，说明有人或东西最近下去过 |
