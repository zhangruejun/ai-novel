# 写作风格参考

## 战斗描写参考

### 示例：试探阶段
```
李长风握紧剑柄，脚步微沉。
对面那人冷哼一声，刀光一闪，直劈而来。
这一刀看似简单，却封住了李长风所有退路。
他侧身避让，剑尖在地上划出一道白痕。
好快的刀。李长风心中一凛。
```

### 示例：分析阶段
```
李长风连退三步，剑势被压得喘不过气。
但他没有慌。
三招了。对方每次出刀，都是右肩先沉。
这是旧伤。
```

### 示例：反击阶段
```
第四刀 coming。
李长风不再退让。
他左脚前踏，剑从下往上挑，正 against 那人右肩。
刀势一偏，擦着李长风的衣袖飞过。
那人脸色大变。
```

### 示例：决胜阶段
```
李长风没有给他喘息的机会。
剑如游龙，直取中路。
那人仓促横刀格挡，却不知这一剑只是虚招。
真正的杀招，是藏在剑鞘中的第二剑。
```

## 对话风格参考

### 示例：简短有力
```
"好剑。"李长风手指轻抚剑身，剑鸣如龙吟。
林婉儿微微一笑："配你正好。"
```

### 示例：暗含冲突
```
"你不该来这里。"
"我偏要来。"
"你会死的。"
"那也比窝囊一辈子强。"
```

### 示例：体现默契
```
李长风看了她一眼。
林婉儿会意，从药箱中取出伤药。
"还是老地方？"
"嗯。"
```

## 环境描写参考

### 示例：简洁有画面感
```
天快黑了。
山寨的篝火陆续点燃，火光在山道上投下摇晃的影子。
风从谷口灌进来，带着松脂的味道。
```

### 示例：配合情绪
```
雨越下越大。
李长风站在屋檐下，看着雨水顺着瓦当淌下来，连成一条水帘。
他已经在这里站了一个时辰。
```

## 心理描写参考

### 错误示例（空洞）
```
他的内心久久不能平静，心情如同翻江倒海一般。
```

### 正确示例（通过行为体现）
```
李长风盯着手里的剑，很久没说话。
然后他把剑收起来，转身走进雨里。
```

## 转场参考

### 示例：时间转场
```
三天后。
青云宗的山门出现在眼前。
```

### 示例：地点转场
```
与此同时，青云宗后山。
赵无极正在煮茶。
```

## 角色说话风格

### 李长风
- 话不多，但每句都有分量
- 遇到问题先分析，不急于表态
- 关键时刻果断，不拖泥带水

### 林婉儿
- 说话温和，但内藏锋芒
- 善于用问句引导对方
- 危急时刻冷静得出奇

### 张云义
- 直来直去，不善拐弯
- 对认定的人和事极其执着
- 激动时语速加快

### 萧炎凤
- 开口就带刺，但内心柔软
- 不服软，即使处于劣势也要嘴硬
- 被触动底线时会罕见地认真
