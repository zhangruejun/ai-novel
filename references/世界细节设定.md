# 世界细节设定

> 本文档用于丰富第一卷（ch044-ch060）归元城篇的世界观细节。
> 所有设定严格基于已有大纲，不挖新坑，不影响后续剧情。
> 最后更新：2026-04-13

---

## 一、朝廷体系在归元城的痕迹

### 1.1 朝廷与宗门的关系

**大纲原文**（plans/武侠仙侠穿越游戏小说大纲.md:84）：
> "朝廷：控制江湖秩序，但无法完全控制宗门；被玄机子暗中操控"
> "宗门：互相制衡，形成平衡；表面被赵无极操控"
> "朝廷需要宗门的力量维持江湖秩序，宗门需要朝廷的合法性认可"

**第一卷中的体现**：
- 归元宗是朝廷正式册封的"正道宗门"，有朝廷颁发的宗门牌照
- 归元城是归元宗的势力范围，但城内设有朝廷的行政机构
- 朝廷不直接管理宗门事务，但宗门需要在朝廷备案

### 1.2 归元城中的朝廷机构

| 机构 | 职能 | 在文中的体现方式 | 是否影响后续剧情 |
|------|------|----------------|----------------|
| **归元府衙** | 归元城行政管理（税收、治安、户籍） | 城门口的告示牌、城内的府衙建筑、偶尔出现的官差巡逻 | 否，纯背景 |
| **刑察司分衙** | 朝廷刑察司在归元城的分支，监视宗门动向 | 城内有一栋不起眼的建筑挂着刑察司牌子，主角路过时感受到被扫了一眼 | 是，后续第五卷刑察司是重要反派机构，第一卷只需暗示存在 |
| **税务司** | 征收商业税、过路税 | 进城需要缴纳入城费（官府收），客栈需要交营业税 | 否，纯背景 |
| **驿站** | 朝廷通信系统 | 城内主街上有一座驿站，有官方快马进出 | 否，纯背景 |

### 1.3 朝廷痕迹的具体表现

**城门**：
- 城墙上除了归元宗的阵法符文，还有朝廷的"镇国大阵"印记（双龙纹章）
- 城门守卫分两拨：归元宗外门弟子（负责修仙者安全检查）+ 朝廷城防军（负责普通百姓出入管理）
- 入城费：普通人免收，武者收取五文铜钱（朝廷规定），修士凭宗门令牌免收

**城内告示牌**：
- 府衙门口贴有朝廷最新政令（如"严禁私斗""武者需登记备案"等）
- 告示牌上还贴有归元宗的公告（如外门考核通知、宗门招新等）
- 两种公告并列，体现朝廷与宗门并存的格局

**街道巡逻**：
- 白天：归元宗外门执法弟子巡逻（负责武者纠纷）
- 夜间：朝廷城防军巡逻（负责普通治安）
- 两者互不干涉，偶尔有摩擦

**货币体系**：
- 朝廷发行的铜钱/银两是通用货币
- 修士之间有以物易物的习惯（灵石、丹药、材料）
- 归元城内有专门的"修真者集市"，用修士货币交易

## 二、势力范围内民众的生活

### 2.1 归元宗势力对民众的影响

**正面影响**：
- 归元宗保护归元城及周边村落不受妖兽和土匪侵扰
- 归元宗每年开放一次"施药日"，免费发放低级丹药给百姓
- 有天赋的平民子弟可以参加外门考核，改变命运（这是底层百姓的最大希望）

**负面影响**：
- 归元宗的税收（灵石、矿产）由朝廷代征，最终落到百姓头上
- 武者之间的争斗偶尔波及平民（但归元宗有规定：严禁在城内斗法，违者逐出宗门）
- 宗门弟子在城市中享有特权（如优先购买、免排队等）

### 2.2 民众对修炼者的态度

| 群体 | 态度 | 表现 |
|------|------|------|
| 普通百姓 | 敬畏+崇拜 | 看到武者会主动让路，称呼"仙师" |
| 商人 | 讨好+利用 | 给宗门弟子打折，希望通过他们攀附关系 |
| 江湖散修 | 羡慕+嫉妒 | "凭什么他们就能进入宗门，我连考核资格都没有" |
| 武者家庭 | 望子成龙 | 倾家荡产送孩子去武馆，希望参加外门考核 |

### 2.3 城市中的修炼者生态

**修炼者聚集区**：
- 城东：归元宗山门及附属建筑（外门弟子宿舍、杂役堂、演武场）
- 城西：修真者集市（材料、丹药、装备交易）
- 城南：武馆和修炼者客栈（散修聚集地）
- 城北：归元宗内门区域（普通人不让进）

**修炼者服务行业**：
- 材料铺：出售矿石、药草、妖兽材料
- 丹药铺：出售回气丹、金疮药、止血散等低级丹药
- 武器铺：出售凡铁武器和低级灵器
- 修炼者客栈：房间有聚灵阵，比普通客栈贵三倍
- 武馆：教基础拳脚和兵器，学费昂贵

## 三、其他修炼职业和组织

### 3.1 大纲中的职业体系

**大纲原文**（plans/武侠仙侠穿越游戏小说大纲.md:65-69）：
> "技能系统：剑道技能（主修）、内功心法、外家武功、招式组合"
> "修仙体系分支：法修（主修灵力，擅长法术与阵法）、体修（主修肉身，以力破法）"

**第一卷中出现的职业**：
- **剑修**：主角、苏寒（以剑为主要武器）
- **刀修**：血影门杀手（血色长刀）
- **拳修/体修**：石小山（肉身力量强）
- **医修**：林婉儿（医术+某种特殊能力）
- **阵法师**：归元宗护城大阵的维护者（背景提及）

### 3.2 修炼者组织（第一卷可提及的）

| 组织 | 性质 | 与主线关系 | 是否影响后续 |
|------|------|-----------|------------|
| **归元宗** | 正道宗门，第一卷主要舞台 | 核心 | 是 |
| **血影门** | 魔教分支，赵无极的工具 | 核心（已被主角清剿断魂岭据点） | 是（后续还有更大势力） |
| **黑风寨** | 土匪山寨，王家豢养的打手 | 已清剿（ch030） | 否 |
| **王氏商行** | 青城镇的富商家族 | 已覆灭（ch015-020） | 否 |
| **百草堂** | 归元城的丹药连锁商号 | 纯背景，主角买药的地方 | 否 |
| **天工坊** | 归元城的武器铸造商号 | 纯背景 | 否 |
| **江湖散修联盟** | 散修自发组织，互通消息 | 纯背景，主角可能听说过 | 否 |

> **注意**：第一卷不要引入新的有影响力的组织，所有新组织都只能作为背景提及，不能有剧情线。

### 3.3 不同职业修炼者的特征

| 职业 | 外观特征 | 战斗风格 | 社会地位 |
|------|---------|---------|---------|
| 剑修 | 背剑或佩剑，气质凌厉 | 攻高防低，招式精妙 | 最高（剑为百兵之君） |
| 刀修 | 佩刀，气质粗犷 | 攻高血厚，招式霸道 | 中等 |
| 拳修/体修 | 身材魁梧，肌肉发达 | 近战肉搏，防御极高 | 中等偏下（被认为是莽夫） |
| 医修 | 气质温和，携带药箱 | 不擅长战斗，治疗能力极强 | 特殊（各方都尊重） |
| 阵法师 | 气质沉稳，携带阵盘/符纸 | 远程布置，需要准备时间 | 高（稀缺人才） |

## 四、归元城详细布局

### 4.1 城市概览

```
                    北
                     ↑
    ┌────────────────────────────────────┐
    │        归元宗内门（禁入）            │
    ├────────────────────────────────────┤
    │   外门弟子区  │  演武场  │ 杂役堂    │  ← 城北（宗门区）
    ├──────────────┼──────────┼───────────┤
    │              │          │           │
    │   府衙       │  主街    │   驿站     │  ← 城中（行政区）
    │   刑察司分衙 │          │           │
    ├──────────────┼──────────┼───────────┤
    │   武馆区     │  客栈街  │   民居     │  ← 城南（商业/居住区）
    ├──────────────┼──────────┼───────────┤
    │   修真集市   │  百草堂  │   天工坊   │  ← 城西（修炼者商业区）
    ├──────────────┼──────────┼───────────┤
    │   东城门     │  后山    │   灵药园   │  ← 城东（归元宗产业）
    └────────────────────────────────────┘
                     ↓
                    南（通往外界）
```

### 4.2 关键地点说明

| 地点 | 说明 | 剧情用途 |
|------|------|---------|
| **归元宗山门** | 城北，依山而建，气势恢宏 | 外门考核地点 |
| **外门弟子区** | 山腰石屋，简陋但安全 | 主角和张云义住处 |
| **演武场** | 外门弟子修炼和比试的地方 | 外门小比地点 |
| **杂役堂** | 分配杂役任务的地方 | 主角领任务处 |
| **灵药园** | 后山，种植灵草灵药 | 主角被安排的杂役任务（赵无极暗中关注此地） |
| **藏经阁** | 外门弟子可进入一层 | 后续主角寻找功法的地方 |
| **府衙** | 城中行政中心 | 背景提及 |
| **刑察司分衙** | 不起眼的小楼 | 主角路过时感受到被监视（暗示后续） |
| **云来客栈** | 中等客栈，城南 | 主角进城时的落脚点 |
| **修真者集市** | 城西，各种修炼者交易 | 主角购买物资 |

## 五、第一卷（ch044-ch060）剧情修正方向

### 5.1 必须删除的元素

| 元素 | 原因 | 替代方案 |
|------|------|---------|
| ❌ 幽冥教 | 不在大纲中 | 改为"黑莲会"（赵无极的秘密组织代号）或直接不命名 |
| ❌ 白莲教 | 不在大纲中 | 删除林婉儿的组织身份 |
| ❌ 血祭仪式 | 不在第一卷计划中 | 改为赵无极在收集年轻修炼者的真气/精血 |
| ❌ 副宗主是内鬼 | 无此设定 | 改为赵无极直接操控 |
| ❌ 先天剑胚 | 第二卷第280章才出现 | 第一卷完全不提 |

### 5.2 必须修正的时间线

| 项目 | 细纲中写的 | 正确设定 | 
|------|-----------|---------|
| 基础剑法 | ch031-ch044写"大成" | ch045才是"精通→大成"，ch031-ch044应该是"精通" |
| 林婉儿登场 | ch045再次登场 | ✅ 合理（上次ch024） |
| 张云义境界 | 剑师境后期 | ✅ 合理 |
| 主角境界 | ch046突破剑师巅峰 | ✅ 合理（力量体系规划） |

### 5.3 ch044-ch060 修正后的剧情线

**ch044-ch046：离开断魂岭，前往归元城**
- 破译密信，发现赵无极与血影门的勾结证据
- 林婉儿再次登场，提供关于赵无极的线索（保持神秘）
- 主角突破剑师巅峰，进入归元城

**ch047-ch052：归元宗外门考核**
- 报名登记→第一关（骨龄/境界）→第二关（试炼谷实战）→第三关（心性测试）
- 赵虎挑衅被秒杀
- 试炼谷遇苏寒（竞争对手）
- 考核通过，录取外门弟子

**ch053-ch060：外门弟子生活+发现赵无极阴谋**
- 外门日常→灵药园杂役→发现灵药园异常
- 灵药园不是普通的药园——赵无极在此暗中收集某种东西
- 外门小比→与苏寒对决
- 发现赵无极在灵药园的暗中活动线索（不是血祭，而是更模糊的阴谋）

> **关键原则**：第一卷只发现"线索"，不揭开全貌。赵无极的阴谋在第二卷才逐步揭露。

## 六、代入感/真实感增强技巧

### 6.1 生活细节

- 客栈老板会打听客人的来历（"小哥看着面生，是来参加外门考核的？"）
- 茶馆里有说书先生讲江湖轶事（"话说那归元宗赵宗主，可是当世高人..."）
- 街边有卖包子的小贩，看到武者会多送一个（"仙师，您慢用"）
- 夜晚有打更人敲梆子（"天干物燥，小心火烛——"）

### 6.2 修炼者特权

- 武者进店不用排队
- 修炼者客栈有聚灵阵，价格贵三倍
- 宗门弟子穿的道袍是身份象征
- 散修没有宗门令牌，进城会被多问几句

### 6.3 阶级差异

- 外门弟子 vs 内门弟子：内门弟子走在路上，外门弟子要让路
- 宗门弟子 vs 散修：散修连报名考核都要排队，宗门推荐的可以直接通过初审
- 有钱的武者 vs 穷武者：有钱的住聚灵阵客栈，穷的睡大通铺
