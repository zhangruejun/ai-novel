# 禁用语列表

> 使用说明：每次Prompt生成时附带此列表。写作审校时逐项检查。
> 版本：v3.0（审查修正版） | 最后更新：2026-04-09

---

## 一、AI高频连接词（禁用）

> 这些词是AI最明显的痕迹，必须完全避免。

```
然而、与此同时、值得注意的是、不得不提、众所周知、
毫无疑问、显而易见、毋庸置疑、不言而喻、
可以说、换句话说、也就是说、总而言之、
不仅如此、更重要的是、与此同时、反观、
除此之外、总而言之、一言以蔽之、
反观、诚然、无独有偶、
```

## 九、AI高频句式模式（禁用）

> 不仅仅是词汇，某些句式模式也是明显的AI痕迹。

### 对称句式（禁用）
```
他做了A，也做了B
既有C，又有D
不仅...而且...
一方面...另一方面...
```

### 总结句式（禁用）
```
不得不说
不可否认
事实上
实际上
```

### AI习惯用语（禁用）
```
OK、没问题、加油
OK了、没问题了、搞定
好的、收到、明白了（现代感太强）
```

## 十一、AI高频结尾模式（避免）

> AI喜欢用这些方式结尾，要有意识地避免。

```
XX望着远方，久久不语
XX转身离去，背影消失在夜色中
XX嘴角露出一丝微笑，一切才刚刚开始
XX深吸一口气，知道更大的挑战还在等着他
夜还很长，路也很长
XX握紧了拳头，眼中闪过坚定的光芒
```

## 十一半、元叙事禁止（审查修正·新增）

> 角色不可能知道"章节"概念，这是作者/读者的视角。使用即打破第四面墙，对沉浸感是毁灭性的。

| 禁止项 | 示例 | 正确写法 |
|--------|------|---------|
| 用"第X章"作为时间计量 | "第3章有1处异常" | "两个时辰前有1处异常" |
| 用"第X章"作为事件计量 | "从第1章到现在" | "从昨天到现在" |
| 角色知道未来章节信息 | "第10章才揭晓" | "真相尚未揭晓" |
| 角色用"章"作为叙事单位 | "这一章发生了什么" | "这一天发生了什么" |

## 十二、现代用语（禁用）

> 古风世界，不要出现现代用语。

```
OK、NO、YES
加油、给力、牛逼、牛逼啊
没问题、好的、收到
系统、程序、数据（除非是系统相关描写）
逻辑、分析、推理（可以偶尔用，但不要频繁）
机会、风险、挑战（可以偶尔用）
```

---

## 十三、使用建议

1. **审校时逐项检查**：每章完成后，用Ctrl+F搜索以上词汇
2. **替换而非删除**：找到禁用词后，用更具体的描写替换
3. **建立个人禁用语库**：根据自己AI的表现，持续添加新的禁用语
4. **不要过度敏感**：偶尔出现一两个禁用词是正常的，不要因此重写整章
5. **重点关注高频词**：以下是最常见的AI痕迹，必须严格禁止：
   - 然而、与此同时、值得注意的是
   - 嘴角勾起一抹微笑
   - 眼中闪过一丝精光
   - 他心中暗道
   - 倒吸一口凉气

## 十四、常见AI痕迹替换示例

| AI原文（禁用） | 替换建议 |
|-------------|---------|
| 他心中暗道："好厉害的对手。" | 他握紧了剑，盯着对方的刀。 |
| 嘴角勾起一抹冷笑 | 他哼了一声。 |
| 眼中闪过一丝精光 | 他眯起了眼睛。 |
| 倒吸一口凉气 | 他停住了脚步。 |
| 他感到一阵复杂的情绪涌上心头 | 他没有说话。过了很久，才说了一句："走吧。" |
| 然而，事情并没有那么简单 | 事情比想象中复杂。 |
| 与此同时，在归元宗后山 | 归元宗后山。 |
| 一道寒光闪过 | 剑出鞘。 |
