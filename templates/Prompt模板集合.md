# Prompt模板集合

> 使用说明：
> 1. 每次生成内容前，复制对应模板并填写
> 2. **每个Prompt必须包含前情提要模块**（见下方通用结构）
> 3. 分场景生成时，每个场景使用一个Prompt
> 4. 生成后保存至：prompts/volXX/chXXX_场景X.md

---

## 通用结构（所有Prompt共用）

```
你现在是一位有10年武侠写作经验的网文作者，擅长战斗描写和角色塑造。

## 前情提要
【上一章结尾】
[粘贴上一章最后200-300字，让AI知道前情]

【当前状态摘要】
- 李长风：[当前境界]，[身体状态]，[当前位置]
- 林婉儿：[当前境界]，[身体状态]，[当前位置]，[与主角关系阶段]
- [其他重要角色状态]
- [关键物品状态]

【必须遵守的信息】
- [列出3-5条AI最容易忘记或搞错的关键信息]

---

## 章节信息
- 章名：[XXX]
- 核心事件：[一句话概括]
- 情绪基调：[轻松/紧张/悬疑/热血/悲壮]
- 字数要求：[800-1200字/场景]，本章共[X]个场景

---

## 场景[X]：[场景名]
- 地点：[具体地点]
- 时间：[具体时间]
- 出场角色：[列出]
- 核心动作：[描述]
- 对话要点：[描述对话的核心内容或冲突]

---

## 写作风格要求
- 战斗描写：突出剑道特色，强调策略而非数值，有动作细节
- 对话风格：符合角色性格，李长风冷静果断，林婉儿聪明缜密
- 环境描写：简洁有画面感，不要过度堆砌辞藻
- 节奏控制：[快节奏/慢节奏/张弛有度]

## 禁用语
请参考references/禁用语列表.md，以下是最常见的禁用词：
- 然而、与此同时、值得注意的是、不得不提
- 他感到一阵复杂的情绪涌上心头
- 嘴角勾起一抹微笑/冷笑
- 眼中闪过一丝精光/杀意
- 倒吸一口凉气
- 犹如/宛如/如同/仿佛（慎用，可以偶尔使用）

## 范围限制
- 只写到[具体事件]为止
- 不要写[明确不要写什么]
- 不要添加大纲中没有的角色
- 不要推进[某条]感情线
```

## 2. 战斗场景专用Prompt

> 适用于有战斗的章节。填写前删除此说明。

```
[通用结构]

## 战斗场景信息
- 对手信息：[对手名称][实力][武器][特点][弱点]
- 战斗原因：[为什么打起来]
- 战斗地点：[具体环境，是否影响战斗]
- 旁观者：[如有，列出]

## 战斗阶段设计
1. 试探阶段（约300字）：双方交手，互相试探，展示对手实力
2. 交锋阶段（约300字）：对手展示真正实力，主角陷入劣势
3. 分析阶段（约200字）：主角观察对手，发现弱点或规律
4. 反击阶段（约300字）：主角利用弱点，扭转局势
5. 决胜阶段（约200字）：主角施展关键一击

## 战斗特色
- 本场战斗特色：[如"剑意对拼""身法较量""阵法困战"，不要千篇一律]
- 战斗代价：[主角是否受伤/消耗/获得感悟]
- 战斗结果：[胜/负/平/中断]

## 注意事项
- 战斗描写要有具体动作细节，不要只写"你来我往"
- 强调策略和智慧，不只是力量对拼
- 战斗中可以穿插对话，增加戏剧性
```

## 3. 感情场景专用Prompt

> 适用于感情培养、互动的章节

```
[通用结构]

## 感情场景信息
- 感情阶段：[初识/熟悉/暧昧/恋人/生死与共]
- 当前关系温度：[0-100分]
- 场景目的：[建立信任/消除误会/感情升温/确认关系]

## 互动设计
- 互动方式：[对话/共同行动/危机互助/日常相处]
- 感情推进要点：
  1. [如"两人因某事产生分歧"]
  2. [如"对话中暗含关心"]
  3. [如"一个细节体现默契"]

## 角色说话风格参考
【[角色A]典型对话】
"[示例对话1]"
"[示例对话2]"

【[角色B]典型对话】
"[示例对话1]"
"[示例对话2]"

## 注意事项
- 不要直白写"我爱你""我喜欢你"等，用行为和细节体现感情
- 感情推进要自然，不要突然升温
- 通过小动作体现关心（如递药、挡雨、默默跟随等）
```

## 4. 日常互动场景Prompt

> 适用于角色之间的日常互动、闲聊、感情培养

```
[通用结构]

## 日常互动信息
- 场景目的：[角色间建立信任/展示角色性格/铺垫后续剧情/调节节奏]
- 角色关系：[如"李长风与张云义兄弟互动"]
- 互动方式：[切磋武艺/聊天/共同完成任务/做饭/整理物品等]
- 性格展示：[通过日常互动展示角色性格]

## 对话设计
- 话题一：[如"讨论刚才战斗的得失"]
- 话题二：[如"聊起各自的过往"]
- 话题三：[如"谈到未来的计划"]

## 注意事项
- 对话要自然，不要像在接受采访
- 适当加入幽默/温馨的元素
- 日常互动中也要有小冲突或分歧，增加真实感
- 至少有一个细节体现角色之间的默契/关心
```

## 5. 群像戏Prompt

> 适用于多人场景，如门派会议、战斗集结、庆功宴等

```
[通用结构]

## 群像戏信息
- 出场角色（3人以上）：[列出所有角色及其目的/立场]
- 场景核心：[多方势力谈判/宗门大会/武林大会/战前部署]

## 角色分工
- 主导者：[谁在主导讨论/行动，核心立场是什么]
- 反对者：[谁持不同意见，为什么反对]
- 调和者：[谁在调和矛盾，怎么做]
- 观察者：[谁在旁观/暗中分析，注意到了什么]

## 冲突/张力
- 核心冲突/分歧：[描述]
- 冲突的结果/走向：[描述]

## 注意事项
- 每个角色必须有明确的立场和态度
- 对话要有交叉，不要变成轮流发言
- 至少有2-3个角色有突出的性格表现
- 不要平均分配戏份，有主次之分
- 主角必须有明确的立场和行动
```

## 6. 回忆杀Prompt

> 适用于插入角色的回忆、过往经历

```
[通用结构]

## 回忆杀信息
- 触发原因：[什么触发了回忆？看到某物/听到某话/来到某地]
- 回忆内容：[简要描述回忆的内容]
- 回忆长度：控制在200-300字以内

## 回忆处理要求
- 回忆与当前剧情的关联：[这段回忆如何推动当前剧情/揭示信息]
- 回忆的情感基调：[温馨/痛苦/遗憾/愤怒]
- 转场处理：
  - 进入回忆：[如何从现实过渡到回忆，如"他看着那把剑，想起了..."]
  - 结束回忆：[如何从回忆回到现实，如"一声呼唤将他拉回现实..."]

## 注意事项
- 回忆不要过长
- 回忆必须有明确的叙事目的，不要为回忆而回忆
- 回忆的内容必须与前文一致，不能与已写内容矛盾
- 回忆中不要加入新的关键信息（关键信息应在正线中揭示）
```

## 7. 突破/升级场景Prompt

> 适用于角色境界突破、实力提升

```
[通用结构]

## 突破信息
- 突破前境界：[当前境界]
- 突破后境界：[目标境界]
- 突破方式：[闭关修炼/战斗中突破/服用丹药/顿悟]
- 突破地点：[具体地点]

## 突破过程设计
- 突破前兆：[有什么预兆/征兆]
- 突破困难：[遇到了什么阻碍？心魔/能量暴走/外力干扰]
- 突破关键：[是什么帮助突破的？感悟/外力/生死之间]
- 突破表现：[突破时的具体表现，能量变化/天地异象等]

## 突破后
- 新能力/变化：[突破后获得了什么新能力]
- 突破代价：[是否有代价？消耗/受伤/后遗症]
- 旁观者反应：[如有旁观者，他们的反应]

## 注意事项
- 不要写成数据堆砌（如"真气增加了XX倍"）
- 重点描写感悟和成长，不是数值变化
- 突破场景要有情感冲击力
```

## 8. 悬疑/解密场景Prompt

> 适用于角色发现线索、解密、推理

```
[通用结构]

## 悬疑信息
- 悬疑核心：[主角要解开什么谜题]
- 线索列表：
  1. [线索1，在哪发现的，看起来是什么]
  2. [线索2]
  3. [线索3]
- 推理过程：[主角如何分析线索，得出结论]

## 悬疑氛围
- 环境配合：[如"昏暗的密室""泛黄的卷轴""血迹斑斑的地面"]
- 信息揭露节奏：[先揭示什么，保留什么悬念]

## 注意事项
- 不要一次性揭露所有信息，留有余地
- 推理过程要符合逻辑，不要凭空猜测
- 线索必须在前文有铺垫，不要突然出现
- 让读者和主角一起推理，不要直接给出答案
- 结尾要留下新的疑问或悬念
```

## 9. 转场/过渡场景Prompt

> 适用于章节之间的过渡、时间跳跃、地点转换

```
[通用结构]

## 转场信息
- 转场类型：[时间跳跃/地点转换/视角切换]
- 时间跨度：[多少天/月/年]
- 地点变化：[从哪里到哪里]

## 转场处理
- 转场写法：选一种
  1. 环境变化法：通过环境描写暗示时间/地点变化
  2. 角色动作法：通过角色的一个动作/一句话过渡
  3. 时间标注法：直接用"三日后"/"三个月后"等标注
- 转场中包含的信息：[至少一个信息点：角色成长/势力变化/新线索等]

## 转场后
- 转场后快速进入核心事件，不要有过长的铺垫
- 转场后的第一个场景必须有吸引力

## 注意事项
- 转场必须自然，不要突然跳跃
- 转场中至少包含一个信息点
- 转场后快速进入核心事件
- 转场不要超过400字
```

## 10. 高潮场景Prompt

> 适用于重大决战、重要对决、剧情高潮

```
[通用结构]

## 高潮场景信息
- 场景性质：[最终决战/生死危机/重大抉择/真相大白]
- 参与角色：[列出所有参与者]
- 赌注/代价：[如果输了会怎样]

## 情绪设计
- 本场情绪基调：[绝望→希望/愤怒→冷静/悲痛→坚定]
- 情绪锚点：
  1. [场景开头要让读者感到XXX]
  2. [场景中间要让读者感到XXX]
  3. [场景结尾要让读者感到XXX]

## 场景结构
1. 开局（约300字）：建立紧张感，展示双方状态
2. 交锋（约400字）：激烈对抗，一方占据优势
3. 低谷（约300字）：主角陷入绝境，看似失败
4. 转折（约300字）：主角发现机会/领悟/外力相助
5. 决胜（约300字）：关键一击，尘埃落定
6. 余韵（约200字）：战后反应，情感释放

## 注意事项
- 高潮场景要有情感冲击力
- 不要写成单纯的力量对拼，要有智慧和意志的较量
- 战斗中穿插回忆/对话/内心独白，增加层次感
- 结果必须有代价，不能完美无缺
```
