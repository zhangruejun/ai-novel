## 详细改动说明

### ch025 分析线索

**改动前问题**：
- 四件异常以列表形式逐一罗列，像写报告而非小说
- "容器"关键词只是一带而过，没有展开主角的心理反应

**改动后**：
- 主角一边敷药一边梳理线索，把信息拆入动作场景中
- "容器"关键词单独成段，加入主角"低头看自己身体"的动作反应
- 四件异常用短句分隔，节奏更紧凑

**改动片段对比**：

```diff
- 他闭上眼睛，开始在脑海中梳理所有的线索——从穿越到现在，
- 每一件异常的事情都被他翻出来重新审视。
-
- 第一件异常：黑风村的村民。
- 那些站姿僵硬、眼神空洞、循环重复同一句话的村民……
-
- 第二件异常：系统乱码。
- "检测到异常波动……数据源受到干扰……容器状态不稳定……"
- 这三句话透露出的信息量极大……

+ 他一边敷药，一边在脑海中梳理今天发生的四件异常。
+
+ 第一件。
+
+ 他想起离开黑风村那晚看到的村民——站姿僵硬，每隔十秒呼吸一次，
+ 一秒都不差。那种行为模式根本不像是活人，更像是……被设定好程序的傀儡。
+
+ 第二件。
+
+ 他想起火堆旁那三秒的乱码。
+
+ "检测到异常波动……数据源受到干扰……容器状态不稳定……"
+
+ "容器"这个关键词让他心头一震。
+
+ 什么容器？
+
+ 他低头看了一眼自己的身体。
```

### ch026 结拜兄弟

**改动前问题**：
- 张云义讲述灭门惨案时纯口头叙述，没有触发物
- 读者无法直观感受到"灭门"的惨烈

**改动后**：
- 增加触发物：一块被烧得焦黑的木牌，张家祠堂牌位残片
- 木牌边缘有明显刀劈痕迹，暗示人为破坏而非意外火灾
- 张云义把木牌递给李长风，"入手冰凉，像是握着一块石头"

**改动片段**：

```diff
+ 走了一段路之后，张云义从怀中摸出一个东西。
+
+ 那是一块被烧得焦黑的木牌，只有半个巴掌大小，上面还能依稀辨认出
+ "张"字的轮廓。木牌的边缘有明显的刀劈痕迹——不是火烧造成的，
+ 而是人为的。
+
+ "这是我张家祠堂的牌位残片。"
+
+ 李长风接过木牌。入手冰凉，像是握着一块石头。
```

### ch028 截胡千年灵芝

**改动前问题**：
- 突破过程太顺利，缺少危机感
- 读者感受不到突破的风险

**改动后**：
- 增加"灵气失控"危机：灵气太多，经脉承受不住
- 主角的心法"像一条细绳要捆住狂奔的野马"——根本捆不住
- 增加外在表现：周身空气扭曲，树叶和碎石悬浮
- 张云义视角看到异常，增加紧张感

**改动片段**：

```diff
+ 但他低估了千年灵芝的能量。
+
+ 灵气太多了。
+
+ 多到他的经脉根本承受不住。
+
+ 他咬牙运转《基础剑法》心法，试图引导这股洪流。但心法就像一条细绳，
+ 要捆住一头狂奔的野马——根本捆不住。
+
+ "不好……"
+
+ 他在心中暗叫一声。
+
+ 灵气失控了。
+
+ ……
+
+ 更让张云义不安的是——李长风周身的空气在扭曲。
+
+ 那不是热浪，而是灵气外溢的征兆。过量的灵气从他的毛孔中渗出，
+ 在他身体周围形成了一层肉眼可见的气场。气场中的树叶和碎石
+ 被轻轻托起，悬浮在半空中。
```

### ch029 突破之后

**改动1：杀手出现前加环境异常暗示**

```diff
+ 但就在笑声还未落下的时候——
+
+ 李长风的耳朵动了一下。
+
+ 不是脚步声。
+
+ 是更细微的东西。
+
+ 洞口的树叶在动——没有风，但地面上的落叶在轻轻抖动。
+ 像是有什么重物在不远处经过，震动了地面。
+
+ 然后是一只鸟。
+
+ 一只栖息在洞口附近树枝上的灰雀，突然毫无征兆地扑腾着翅膀飞走了。
+ 它飞得很急，像是被什么东西惊扰了。
+
+ 张云义也察觉到了。他按在剑柄上的手微微收紧，侧耳倾听。
+
+ 三息之后——
+
+ "沙沙。"
+
+ 脚步声来了。
```

  疤脸男子点了点头，右手缓缓按上刀柄。

  "上路吧。"他说。
