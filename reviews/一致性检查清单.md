# 一致性检查清单

> 使用说明：每章完成后逐项检查。每10章/50章进行深度检查。
> 复制到：reviews/chXXX_检查清单.md

---

## 每章完成后检查

### 角色一致性
- [ ] 角色名字是否正确（李长风、林婉儿、张云义等）
- [ ] 角色当前境界是否正确
- [ ] 角色当前状态是否正确（是否有伤、是否在附近等）
- [ ] 角色性格表现是否符合设定
- [ ] 角色称呼是否一致（如张云义叫主角"大哥"等）
- [ ] 角色的说话风格是否符合其性格/身份/背景
- [ ] 角色的行为是否符合其当前动机
- [ ] 角色是否"消失又突然出现"

### 时间线一致性（新增）
- [ ] 当前章节的"Day X"是否与时间线管理文档一致
- [ ] 角色的年龄是否与时间线推算一致
- [ ] 季节/天气是否与时间线设定的季节对应
- [ ] 路程耗时是否合理（参考时间线管理中的路程耗时表）
- [ ] 角色修炼/闭关时间是否与外界流逝时间一致
- [ ] 任务时限是否按时计算（如新手任务截止日）
- [ ] 是否有角色"凭空消失或出现"（时间跳跃未说明）
- [ ] 时间推进是否有明确标记（如"三日后""次日"等）

### 剧情一致性
- [ ] 时间线是否正确（是否跳跃/倒流）
- [ ] 地点是否正确（角色是否在合理的位置）
- [ ] 前文提到的物品/信息是否被正确使用
- [ ] 伏笔是否被正确处理（埋设/回收）
- [ ] 反派行动是否符合其动机
- [ ] 剧情推进是否合理（没有突然跳跃或倒退）
- [ ] 本章内容是否与大纲一致（没有跑偏）
- [ ] 与上一章的衔接是否自然流畅

### 力量体系一致性
- [ ] 境界等级是否正确（不要出现大纲中没有的境界）
- [ ] 战斗结果是否合理（越级战斗是否有合理解释）
- [ ] 技能使用是否符合设定
- [ ] 战斗中的能量/真气消耗是否合理
- [ ] 角色的战斗力表现是否符合其当前境界
- [ ] 突破升级是否有合理的铺垫和条件

### 战斗合理性（新增）
- [ ] 战斗描写是否有具体动作细节（不是"你来我往"等空话）
- [ ] 战斗是否有策略和智慧（不是单纯力量对拼）
- [ ] 战斗结果是否有合理的代价（受伤/消耗/感悟）
- [ ] 战斗特色是否与其他战斗有区别（不是千篇一律）
- [ ] 越级战斗是否有合理解释（系统/剑道天赋/环境优势）

### 语言质量
- [ ] 是否有禁用AI词汇（参考禁用语列表）
- [ ] 是否有现代用语（应使用古风用语）
- [ ] 对话是否符合角色性格
- [ ] 是否有重复的句式或描写
- [ ] 是否有过度空洞的心理描写
- [ ] 是否有"注水"内容（无意义的描写/重复内容）
- [ ] 是否有过度发挥内容（写了大纲之外的剧情）
- [ ] 说书人/旁白用语是否过度使用（却说、且说等）
- [ ] 是否有对称句式（他做了A，也做了B；既有C，又有D）

### 伏笔呼应（新增）
- [ ] 本章是否埋设了新伏笔（如有计划）
- [ ] 本章是否呼应了前文的伏笔（如有需要）
- [ ] 本章是否回收了旧伏笔（如有计划）
- [ ] 伏笔处理是否自然（不是生硬插入）

### 节奏检查
- [ ] 本章是否有明确的核心事件
- [ ] 场景转换是否自然
- [ ] 结尾是否有钩子
- [ ] 字数是否在2800-3200字范围内
- [ ] 本章节奏是否符合预期（快/慢/张弛有度）

### 结构检查
- [ ] 本章结构与上一章是否有明显雷同（模板化检测）
- [ ] 本章是否使用了之前没用过的新鲜写法
- [ ] 本章是否有至少一个亮点（好对话/好描写/好转折）
- [ ] 场景类型是否与上一章有变化（不要连续3章同一类型）

## 每10章深度检查

### 时间线深度检查（新增）
- [ ] 对照`references/时间线管理.md`，确认当前章节的Day X标记正确
- [ ] 确认所有出场角色的年龄与时间线一致
- [ ] 确认季节/天气与时间线设定的季节对应
- [ ] 确认路程耗时合理（十里铺→黑风寨→青州城等）
- [ ] 确认修炼/闭关时间与外界流逝一致
- [ ] 确认系统任务时限按时计算
- [ ] 确认没有时间跳跃未说明的情况
- [ ] 更新角色年龄（如果跨越生日或新年）
- [ ] 对照修炼进度时间线，确认境界达成时间合理

### 剧情进度
- [ ] 当前进度是否符合卷级规划
- [ ] 关键节点是否按时完成
- [ ] 是否有偏离大纲的剧情
- [ ] 剧情节奏是否有起伏（高潮-平淡-高潮）
- [ ] 过去10章中是否有"水章节"（无核心事件的章节）

### 角色成长
- [ ] 主角成长是否符合预期
- [ ] 配角戏份是否合理（不要边缘化重要配角）
- [ ] 感情发展节奏是否正确（不要过快也不要过慢）
- [ ] 是否有角色"消失"太久（超过20章未出场）
- [ ] 角色的动机是否始终清晰

### 伏笔追踪
- [ ] 已埋设的伏笔列表：[列出所有未回收伏笔]
- [ ] 计划回收的伏笔：[列出计划在哪些章节回收]
- [ ] 是否有遗漏需要回收的伏笔
- [ ] 伏笔的埋设与回收间隔是否合理（不要太快也不要太久）
- [ ] 是否有伏笔超过50章未回收（需要紧急处理）

- [ ] 反派是否有合理戏份（不要长期不出场）
- [ ] 反派行动是否符合其逻辑
- [ ] 反派是否有成长/变化

### 读者体验
- [ ] 连续3章是否有相似结构（模板化检测）
- [ ] 是否连续5章没有高潮（节奏检测）
