## 核心问题诊断

**不是战斗分几章写的问题，而是战斗弧线背靠背的问题。**

当前结构（问题版）：
```
[战斗弧线1: 5章] ch032-ch036 ← 探子/分舵
[缓冲: 1章]     ch037       ← 只1章喘息！
[战斗弧线2: 5章] ch038-ch042 ← 杀手/血屠 ← 背靠背！
[缓冲: 3章]     ch043-ch045 ← 破译/林婉儿/突破
[缓冲偏淡: 2章] ch046-ch047 ← 进城/报名
[战斗弧线3: 1章] ch048       ← 赵虎秒杀
[缓冲: 1章]     ch049       ← 考核第一关
[战斗弧线4: 2章] ch050-ch051 ← 试炼谷+苏寒 ← 背靠背！
[缓冲: 3章]     ch052-ch054 ← 心性/分配/黑色花朵
[悬疑弧线: 2章] ch055-ch056 ← 密室/破译
[缓冲: 2章]     ch057-ch058 ← 合作/房间被翻
[战斗弧线5: 2章] ch059-ch060 ← 小比+决赛
```

**问题**：
1. 弧线1和弧线2之间只有1章缓冲（ch037），读者刚喘口气又打架
2. 试炼谷+苏寒两场背靠背（ch050-ch051）
3. 缓冲章太"纯"——只是过渡，没有爽点

---

## 重新设计的节奏结构

### 设计原则
1. **一场战斗弧线 = 起因(1章) + 发展(1-2章) + 高潮(1章) + 收尾(1章) = 4-5章**
2. **战斗弧线之间 = 至少3章缓冲，其中至少1章有非战斗爽点**
3. **缓冲章不能纯过渡——必须有信息碾压/悬念/升级/角色互动等爽点**

**曲线特征**：
- 每个战斗弧线后有明显的"情绪回落"（缓冲段）
- 缓冲段不是平的——有小幅的"非战斗爽点"波峰
- 整体趋势向上（越到后面越紧张）
- 弧线3和弧线5是紧凑短弧线（2章），不需要长时间缓冲

## 总结

### 改动核心
1. **弧线1和弧线2之间从1章缓冲→4章缓冲**（解决背靠背问题）
2. **所有缓冲章都有非战斗爽点**（信息碾压/悬念/升级/角色互动/掌控布局）
3. **试炼谷+苏寒合并为2章紧凑弧线**（可接受）
4. **ch031开篇直接遇袭**（提升开篇冲击力）
5. **赵虎秒杀独立为打脸爽点**（不属于任何弧线）

### 预期效果
- 战斗弧线之间缓冲从1-2章→平均2.8章
- 缓冲章爽点覆盖率从30%→93%
- 读者体验从"又打架"→"打架-爽-烧脑-升级-悬念-再打架"
