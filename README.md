## 快速开始

### ⚠️ 第一步：必读文档（按顺序，缺一不可）

**⚠️ 如果你是AI实例（包括新对话），必须先读完写作执行手册再执行任何操作！**

1. **必读1**：[`写作执行手册.md`](写作执行手册.md) — ⚠️ **最重要！包含7步流程和铁律，违反任何一条等于任务失败**
2. **必读2**：[`plans/工作流.md`](plans/工作流.md) — 详细工作流程
3. **必读3**：[`plans/AI写作控制指南.md`](plans/AI写作控制指南.md) — 写作技巧和避坑指南
4. 最后读：[`plans/武侠仙侠穿越游戏小说大纲.md`](plans/武侠仙侠穿越游戏小说大纲.md) — 了解完整大纲

### 第二步：开始写作第1章

1. **制作细纲**：
   - 复制[`templates/章级细纲模板.md`](templates/章级细纲模板.md)到`outlines/vol01/ch001_细纲.md`
   - 根据大纲第一卷内容填写

2. **生成Prompt**：
   - 从[`templates/Prompt模板集合.md`](templates/Prompt模板集合.md)中选择合适的模板
   - 填入细纲内容
   - 附带角色状态卡、禁用语列表

3. **AI生成**：
   - 分场景生成（每场景800-1200字）
   - 保存到`drafts/vol01/`

4. **人工审校**：
   - 使用[`reviews/一致性检查清单.md`](reviews/一致性检查清单.md)逐项检查
   - 修改AI生成的不合理内容
   - 润色语言，去除AI痕迹

5. **保存定稿**：
   - 保存到`chapters/第001章_章名.md`
   - 更新[`references/角色状态卡.md`](references/角色状态卡.md)、[`references/伏笔追踪表.md`](references/伏笔追踪表.md)
   - 填写[`logs/每日写作日志.md`](logs/每日写作日志.md)

### 第三步：持续写作

- 每天写2-3章（约6000-9000字）
- 每周进行一次维护
- 每10章进行一次深度检查
- 每50章进行一次全面审查

## 核心原则

1. **AI是执行者，不是创作者** — 你提供所有创意和结构，AI负责填充文字
2. **细节决定成败** — 给AI的指令越详细，输出质量越高
3. **人工审校不可少** — AI生成的内容必须经过人工检查和修改
4. **建立系统化管理** — 角色状态、伏笔追踪、剧情进度都要记录在案
5. **持续优化** — 根据AI的表现不断调整Prompt和流程
6. **保持个人风格** — 人工润色环节是注入个人风格的关键步骤
7. **开始写比写得好更重要** — 不要等所有文档完美了才开始写
8. **写作执行手册铁律** — ⚠️每章必须分2-4个场景，每场景独立Prompt和草稿，写完必须更新状态文档（角色卡、伏笔表、日志）

## 常见问题FAQ

### Q1: 每天应该写多少章？

建议每天2-3章。新手刚开始可能只能写1章，这是正常的。随着熟练度提升，速度会加快。

### Q2: AI写的内容不满意怎么办？

1. 调整Prompt，给出更具体的指令
2. 换用备用AI模型
3. 如果多次生成都不满意，可以自己写这一段
4. 不要在一章上死磕，先写下去，后面再回头修改

### Q3: 写到后面忘记前面的设定怎么办？

这就是为什么需要`references/角色状态卡.md`和`reviews/伏笔追踪表.md`。每章完成后更新这些文档，写作前查阅它们。

### Q4: 如何保持章节之间的一致性？

1. 写作前快速阅读上一章结尾（300-500字）
2. 查阅角色状态卡确认当前角色状态
3. 使用一致性检查清单逐项检查
4. 每10章进行一次深度检查

### Q5: 多久需要存稿？

建议始终保持3-5章的存稿。这样即使某天无法写作，也有内容发布。

### Q6: AI写战斗总是千篇一律怎么办？

在Prompt中指定本场战斗的独特特色：
- 不是力量对拼，而是身法较量
- 重点是智谋，不是实力
- 利用环境优势
- 有旁观者反应

### Q7: 如何去除AI痕迹？

1. 使用禁用语列表（Ctrl+F搜索替换）
2. 替换空洞的心理描写为行为描写
3. 调整句式和节奏
4. 注入个人风格和细节
